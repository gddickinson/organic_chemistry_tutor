# Phytohormones in depth

Phytohormones — endogenous, low-concentration,
mobile signal molecules — coordinate plant
development + responses to environment.  Eleven
hormone classes are now recognised.  This lesson
gives the signalling architecture + key clinical /
agricultural applications for each.

For per-class catalogue + structures + agonists +
antagonists, see **BT-2.0 "Plant hormones" tab**.

## Auxin — the master gradient hormone

Major natural auxin: **indole-3-acetic acid (IAA)**.

### Functions

- Apical dominance (suppresses axillary buds).
- Tropisms (phototropism, gravitropism).
- Vascular differentiation.
- Lateral root + adventitious root formation.
- Fruit set + parthenocarpy.
- Embryo apical-basal patterning.
- Rooting of cuttings.

### Polar transport

Auxin is uniquely transported in a directional
manner via:
- **PIN proteins** (efflux carriers; polarised on
  cell membrane → directional flux).
- **AUX1 / LAX** (influx carriers).
- **ABCB / PGP** (auxiliary efflux).

PIN polarisation is dynamically reorganised by
auxin itself (positive feedback) → auxin canals
form in tissues, channeling flux toward sinks
(leaf primordia, vascular strands).

### Perception + signalling

- **TIR1 / AFB receptors** — F-box proteins; bind
  auxin → Aux / IAA repressor proteins +
  ubiquitinate them → 26S-proteasome degradation.
- Aux / IAA gone → ARF transcription factors freed
  → ARF-target genes (cell wall, division, growth)
  expressed.

A second perception system involves **ABP1** at the
plasma membrane (rapid responses, electrical, etc.)
— still controversial in some details.

### Synthetic auxins + applications

- **2,4-D** — herbicide (selective for dicots).
- **IBA + NAA** — rooting hormones (cuttings).
- **Picloram** + **dicamba** + **clopyralid** —
  herbicides + plant growth regulators.

## Cytokinin

Major natural cytokinins: **trans-zeatin (tZ),
isopentenyladenine (iP)**.

### Functions

- Cell division promotion (tissue culture).
- Shoot-meristem maintenance.
- Apical-dominance antagonism.
- Leaf senescence delay.
- Root-to-shoot signalling of N status.
- Chloroplast biogenesis + greening.
- Vascular development.

### Synthesis + transport

- **IPT** (isopentenyltransferase) attaches
  isopentenyl group to AMP → cytokinin precursors.
- **CKX** (cytokinin oxidase) catabolises.
- Synthesised in roots primarily; transported
  acropetally via xylem to shoots.

### Perception + signalling

A two-component system co-opted from bacteria:
- **AHK receptors** (membrane histidine kinases) —
  bind cytokinin → autophosphorylate.
- **AHP** (His-containing phosphotransfer) — relay
  to nucleus.
- **ARR** (response regulators) —
  - Type-B ARRs are TFs activating cytokinin-
    response genes.
  - Type-A ARRs are negative-feedback repressors.

### Applications

- Tissue culture (auxin / cytokinin ratio decides
  callus → root vs shoot regeneration; Skoog +
  Miller 1957).
- Crop yield improvements (manipulating CKX → seed
  yield in rice).

## Gibberellin (GA)

> 130 GAs identified; only ~ 4-5 are bioactive
(GA1, GA3 / gibberellic acid, GA4, GA7).

### Functions

- Stem elongation (dramatic effect).
- Seed germination (breaks dormancy).
- Floral induction.
- Fruit set + parthenocarpy.
- Mobilisation of seed reserves (α-amylase
  induction in cereal aleurone).

### Perception + signalling

- **GID1 receptor** — soluble; binds GA → bound to
  DELLA repressor proteins → recruits SCF-SLY1 /
  GID2 E3 ligase → DELLA ubiquitinated + degraded.
- **DELLAs** (RGA, GAI, RGL1-3) released →
  transcriptional reprogramming.

### Green Revolution

The 1960s yield doubling of wheat + rice via
**semi-dwarf** varieties:
- Norman Borlaug's wheat: **Rht** alleles =
  GA-INSENSITIVE DELLA mutations.
- IR8 rice: **sd1** = GA20 oxidase loss-of-function
  (less bioactive GA).

Shorter stems → less lodging → higher harvest
index → higher yields.  Borlaug 1970 Nobel Peace.

### Synthetic GAs + applications

- **GA3** — seedless grape size, brewing barley
  malting, breaking dormancy.
- **Paclobutrazol + uniconazole** — GA biosynthesis
  inhibitors; turfgrass + ornamentals dwarfing.

## Abscisic acid (ABA)

The drought + stress hormone.

### Functions

- Stomatal closure under drought.
- Seed dormancy maintenance.
- Embryo desiccation tolerance.
- Senescence (despite name "abscisic" — ethylene
  is the actual abscission hormone).
- Cold + salt stress tolerance.

### Perception + signalling

The 2009 mechanism breakthrough:
- **PYR / PYL / RCAR receptors** — soluble;
  bind ABA + recruit phosphatase **PP2C**.
- PP2C inhibited → can no longer dephosphorylate +
  inactivate **SnRK2 kinases**.
- Active SnRK2 → phosphorylate downstream targets
  (SLAC1 anion channel for stomatal closure;
  ABF / AREB transcription factors for stress
  gene expression).

### Applications

- Pre-harvest stress treatments → improved drought /
  cold tolerance.
- ABA-mimic compounds (pyrabactin) as research +
  potential agrochemical tools.

## Ethylene — the gaseous hormone

The smallest hormone.  Discovered when 19th-century
gas-lamp leaks caused leaf abscission near street
lamps.

### Functions

- Fruit ripening (climacteric fruits — banana,
  apple, tomato, avocado).
- Senescence + leaf abscission.
- Triple response in dark-grown seedlings (shorter
  + thicker hypocotyl + apical hook exaggeration).
- Flower senescence.
- Stress responses (flooding-induced shoot
  elongation in rice + lotus).
- Sex determination in cucurbits.

### Synthesis

Methionine → SAM → ACC (1-aminocyclopropane-1-
carboxylic acid) → ethylene + CO2 + cyanide.

Catalysed by:
- **ACS** (ACC synthase) — rate-limiting.
- **ACO** (ACC oxidase) — final step, requires O2 +
  Fe + ascorbate.

### Perception + signalling

- **ETR1 / ERS1 / EIN4 receptors** — endoplasmic-
  reticulum-localised His kinases; in absence of
  ethylene actively SUPPRESS the response pathway.
- Ethylene binding INACTIVATES receptors → CTR1
  kinase inactivated → EIN2 dephosphorylated → EIN3
  / EIL TFs accumulate in nucleus → ERF
  transcription factor cascade → response genes.

### Applications

- **Ethrel (ethephon)** — chemical that releases
  ethylene; used to ripen tomatoes + fruit
  uniformly.
- **1-MCP (1-methylcyclopropene)** — ethylene-
  receptor antagonist; preserves cut flowers + fresh
  fruit shelf life (SmartFresh).
- **AVG (aminoethoxyvinylglycine)** — ACC synthase
  inhibitor; pre-harvest spray to delay apple
  ripening.

## Brassinosteroids (BRs)

Polyhydroxylated steroids; brassinolide most active.

### Functions

- Cell elongation + division.
- Vascular differentiation.
- Photomorphogenesis modulation.
- Stress tolerance (cold, salt, drought).
- Senescence.
- Reproductive development.

### Perception + signalling

- **BRI1 receptor kinase** at the plasma membrane —
  leucine-rich-repeat receptor-like kinase (LRR-
  RLK).  Binds BR + complexes with co-receptor
  BAK1.
- BSK1 + CDG1 kinases activated → BSU1 phosphatase
  → dephosphorylates + inactivates BIN2 (a GSK3-
  like kinase).
- BIN2-inactivated → BES1 + BZR1 transcription
  factors dephosphorylated + active → BR-response
  gene expression.

### Applications

- Crop-improvement target — BR-tuning cultivars
  with altered architecture.
- Developmental biology research probe.

## Jasmonates (JA + JA-Ile)

### Functions

- Wounding + insect-defence signalling.
- Stamen + pollen development.
- Trichome induction.
- Senescence.
- Root growth modulation.

### Perception + signalling

- **JA-Ile** (the bioactive form) binds **COI1 F-box**
  receptor.
- COI1-JA-Ile complex recruits **JAZ repressors** →
  ubiquitinated + degraded.
- JAZ-repressed transcription factors (MYC2 + ICE1
  + ERFs) released → defence + developmental gene
  expression.

A direct mechanistic parallel to auxin TIR1
(F-box-receptor + repressor-degradation paradigm).

### Applications

- Plant-defence priming via JA-mimic sprays
  (commercial use limited).

## Salicylic acid (SA)

### Functions

- Systemic acquired resistance (SAR) to biotrophic
  pathogens.
- Heat-induced flowering (in some species).
- Thermogenesis (skunk cabbage Arum heating).

### Perception + signalling

- **NPR1** — master regulator; oligomeric in
  cytoplasm; SA binding → monomerises → enters
  nucleus → interacts with TGA TFs → PR
  (pathogenesis-related) gene expression.
- **NPR3 / NPR4** — also SA receptors; turnover
  control of NPR1.

### Applications

- **BTH (Acibenzolar-S-methyl, Actigard)** — SA
  mimic; commercial plant-defence elicitor.
- The aspirin connection — willow-bark salicin →
  salicylic acid → metabolised; basis of the human
  drug.

## Strigolactones

Discovered in the 2000s as the missing hormone.

### Functions

- Inhibit shoot branching.
- Promote arbuscular mycorrhizal symbiosis (root-
  exudate signal).
- Trigger Striga + Orobanche parasitic-plant seed
  germination (the original discovery context).
- Control root architecture.

### Perception + signalling

- **D14 receptor** — α/β-hydrolase fold; binds
  strigolactone + cleaves it (covalent intermediate).
- Recruits **MAX2 F-box** → ubiquitinates SMXL /
  SMAX1 repressors → degradation → branching-
  suppression genes expressed.

### Applications

- **GR24** — synthetic strigolactone; research +
  potential application as suicidal-germination
  inducer to clear *Striga* parasitic-weed seed
  banks.

## Peptide hormones

A growing class, including:
- **CLE (CLAVATA3-like)** — meristem-size
  regulation.
- **EPF (epidermal patterning factor)** — stomatal
  density.
- **RALF (rapid alkalinisation factor)** — pollen-
  tube reception, pH signalling.
- **PSK + PSY (phytosulfokine)** — growth.
- **CEP (C-terminally encoded peptides)** — N-status
  signalling root → shoot.
- **IDA (inflorescence deficient in abscission)** —
  abscission control.

Peptide-hormone perception is by LRR-RLKs +
RALF-FERONIA system.

## Other signal classes

- **Karrikins** — smoke-derived butenolides; trigger
  germination of fire-adapted species.
- **Polyamines** (spermine, spermidine, putrescine)
  — growth + stress.
- **NO (nitric oxide)** — multiple stress + signal
  roles.
- **H2S, H2O2** — secondary messengers.
- **Volatile organic compounds (VOCs)** — induced
  systemic resistance + airborne signalling between
  plants.

## Hormone crosstalk

Plant biology is dominated by hormone CROSSTALK +
combinatorial signalling:
- Auxin / cytokinin ratio determines tissue
  regeneration outcome.
- Auxin / strigolactone for branching control.
- ABA / GA antagonism for seed-dormancy break.
- Ethylene / auxin in fruit set.
- JA / SA antagonism between necrotroph + biotroph
  defence.
- BR / GA + BR / JA synergies in growth-defence
  trade-offs.

A modern view treats the plant phytohormone network
as an integrated control system rather than a
collection of independent pathways.
