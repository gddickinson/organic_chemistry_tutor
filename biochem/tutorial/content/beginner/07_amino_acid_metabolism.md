# Amino-acid metabolism

Amino acids are the building blocks of proteins,
carbon skeletons for gluconeogenesis + ketogenesis,
nitrogen donors for nucleotides + heme, and
precursors for neurotransmitters, hormones, and
specialised metabolites.  Their metabolism +
nitrogen disposal underlie a large fraction of
inborn errors of metabolism (IEMs).

## The 20 + 2 amino acids

The genetic code encodes 20 standard amino acids
+ 2 specialty additions:
- **Selenocysteine (Sec)** — encoded by UGA
  (recoded with SECIS element); selenoproteins
  (GPx, TrxR, deiodinase).
- **Pyrrolysine (Pyl)** — UAG-recoded; some
  archaea + bacteria; not in mammals.

### Essential vs non-essential

For humans:
- **Essential** (must be obtained from diet):
  His, Ile, Leu, Lys, Met, Phe, Thr, Trp, Val.
- **Conditionally essential**: Arg (insufficient
  in trauma / illness / preterm), Cys (Met-
  derived), Gln (severe stress), Tyr (Phe-
  derived), Pro (synthesised but high needs in
  burns / wound healing), Gly.
- **Non-essential**: Ala, Asn, Asp, Glu, Ser,
  Pro (mostly).

## Synthesis

Carbon skeletons of non-essential amino acids
come from glycolytic + TCA intermediates +
PPP outputs:

| Amino acid | Precursor |
|------------|-----------|
| Ala | Pyruvate (transamination from Glu) |
| Asp | Oxaloacetate |
| Glu | α-KG |
| Gln | Glu + NH4⁺ (ATP-dependent) |
| Asn | Asp + NH4⁺ (ATP-dependent) |
| Ser | 3-PG |
| Gly | Ser (SHMT) or oxidative cleavage of glycine |
| Cys | Met (transsulfuration) + Ser |
| Tyr | Phe (PAH; covered below) |
| Pro | Glu (P5C cycle) |
| Arg | (mostly diet; partial urea-cycle synthesis) |
| His | (essential — diet) |

Plant + microbial amino-acid biosynthesis is more
extensive (covered in **BT-3.0 advanced "Plant
secondary metabolism"** + **MB-4.0 future "Microbial
metabolism"**).

## Transamination

The dominant amino-acid catabolic + interconversion
reaction:

amino acid + α-KG ⇌ α-keto acid + Glu

Catalysed by **aminotransferases (AT)** = "trans-
aminases":
- **PLP (vitamin B6) prosthetic group** — forms
  Schiff base with substrate.
- **AST (aspartate AT)** = GOT — Asp + α-KG ⇌ OAA + Glu.
- **ALT (alanine AT)** = GPT — Ala + α-KG ⇌ Pyr + Glu.

ALT + AST are clinical liver-injury markers
(elevated in viral hepatitis, drug-induced liver
injury, alcoholic + non-alcoholic fatty liver,
muscle injury).

## The urea cycle

Mammals dispose of nitrogen as urea (less toxic,
less water-demanding than NH₃; less wasteful than
uric acid).

Five enzymes, partly mitochondrial + partly
cytosolic:

1. **CPS-I** (carbamoyl phosphate synthetase I) —
   mitochondrial; NH₃ + HCO₃⁻ + 2 ATP → carbamoyl
   phosphate.  RATE-LIMITING.  N-acetylglutamate
   activator.
2. **OTC** (ornithine transcarbamylase) —
   mitochondrial; carbamoyl-P + ornithine →
   citrulline.
3. **ASS** (argininosuccinate synthetase) —
   cytosolic; citrulline + Asp + ATP →
   argininosuccinate.
4. **ASL** (argininosuccinate lyase) —
   argininosuccinate → arginine + fumarate.
5. **Arginase** — arginine → urea + ornithine.

Urea-cycle defects:
- **OTC deficiency** — most-common; X-linked
  recessive (~ 50 % of UCD).  Severe neonatal
  hyperammonaemia in males; symptomatic
  carriers in females.
- **ASS / ASL / arginase / NAGS deficiencies**
  — autosomal recessive.

Treatment: dietary protein restriction; arginine
+ citrulline supplementation; **ammonia
scavengers** (Na phenylbutyrate / Na benzoate /
glycerol phenylbutyrate) for chronic management;
**carglumic acid** for NAGS deficiency; emergency
dialysis + IV scavengers + dextrose (with insulin
to drive protein synthesis).

## Glucogenic vs ketogenic

Amino-acid catabolism funnels carbons to
intermediates that are either glucogenic
(can become pyruvate / OAA / glucose), ketogenic
(acetyl-CoA / acetoacetate → ketones), or both:

- **Purely ketogenic**: Leu, Lys.
- **Both**: Ile, Phe, Trp, Tyr, Thr.
- **Purely glucogenic**: all others (Ala, Arg,
  Asn, Asp, Cys, Glu, Gln, Gly, His, Met, Pro,
  Ser, Val).

In starvation, glucogenic AAs from muscle
proteolysis fuel hepatic gluconeogenesis (the
glucose-alanine cycle: muscle Ala → liver
gluconeogenesis → glucose → muscle).

## One-carbon metabolism

Folate-dependent transfers of methyl + methylene +
methenyl + formyl groups for:
- Purine synthesis (covered in **BC-4.0 beginner
  "Nucleotide metabolism"**).
- Thymidylate synthesis (dUMP → dTMP via
  thymidylate synthase + 5,10-methylene-THF).
- Methionine cycle: Hcy → Met (methyltransferase
  + B12).
- SAM regeneration → methylation reactions
  (DNA, histones, neurotransmitters,
  phospholipids).

Folate deficiency → megaloblastic anaemia +
neural-tube defects (preconception folic acid
recommended).
B12 deficiency → megaloblastic anaemia + funicular
myelopathy + peripheral neuropathy.

## Specialised pathways + IEMs

### Phenylalanine + tyrosine

- **PAH (phenylalanine hydroxylase)** — Phe → Tyr;
  BH4 (tetrahydrobiopterin) cofactor.
- **PKU** (phenylketonuria) — PAH deficiency;
  Phe accumulates → neurotoxic.  Newborn-screened;
  Phe-restricted diet from birth.  Sapropterin
  (BH4) for some BH4-responsive variants.
  Recently approved **pegvaliase** (PEGylated
  phenylalanine ammonia-lyase enzyme) for adults.
- **Tyrosinaemia type I** — fumarylacetoacetate
  hydrolase deficiency; nitisinone treatment.
- **Alkaptonuria** — homogentisate dioxygenase
  deficiency; black urine + ochronosis.
- **Albinism** — tyrosinase deficiency.

### Branched-chain amino acids (BCAAs)

- **MSUD (maple syrup urine disease)** — branched-
  chain α-ketoacid dehydrogenase deficiency;
  characteristic urine odour; severe neonatal
  encephalopathy.
- Treatment: BCAA-restricted diet; thiamine for
  E2-responsive variants; liver transplant for
  classic.
- **Isovaleric acidaemia + propionic acidaemia +
  methylmalonic acidaemia** — organic acidaemias
  in BCAA breakdown; metabolic crises.

### Methionine + homocysteine

- **Homocystinuria** — cystathionine β-synthase
  (CBS) deficiency; AR; Marfan-like phenotype +
  thrombosis + intellectual disability.  B6
  (PLP) responsive in some variants; betaine +
  folate + B12 for others.
- **MTHFR** polymorphisms (677C>T) — mild
  homocysteine elevation; cardiovascular-risk
  association debated.

### Other notable IEMs

- **Hartnup disease** — neutral-amino-acid
  transporter (SLC6A19) defect; pellagra-like
  rash + ataxia.
- **Cystinuria** — basic amino-acid transporter
  defect; cystine kidney stones.
- **Lysinuric protein intolerance** — diabasic-
  amino-acid transporter; affects urea cycle.
- **Glycine encephalopathy (NKH)** — glycine
  cleavage system deficiency; severe neonatal.

## Specialised products

### Neurotransmitters + amines

- **Glutamate** — excitatory NT.
- **GABA** — Glu decarboxylated by GAD.
- **Glycine** — inhibitory NT.
- **Dopamine + noradrenaline + adrenaline** —
  Tyr → DOPA → dopamine → NE → adrenaline.
- **Serotonin** — Trp → 5-HTP → 5-HT.
- **Histamine** — His decarboxylated.
- **Melatonin** — 5-HT → N-acetyl-5-HT →
  melatonin (pineal).

### Other amino-acid-derived products

- **Heme** — Gly + succinyl-CoA → δ-ALA → ...
  → heme.  Defects → porphyrias.
- **Creatine** — Arg + Gly + Met → creatine →
  creatine-phosphate (muscle energy buffer).
- **Glutathione** — γ-Glu-Cys-Gly tripeptide.
- **Polyamines** — Orn → putrescine →
  spermidine + spermine.
- **Nitric oxide** — Arg + O₂ → NO + citrulline
  (NOS family).
- **Carnitine** — Lys + Met-derived; β-oxidation
  shuttle.
- **Thyroid hormones** — Tyr-iodination on
  thyroglobulin.

## Protein turnover

Proteins are constantly synthesised + degraded:
- Cellular protein t1/2 ranges from minutes
  (cyclins, MYC, many TFs) to days (histones,
  haemoglobin) to weeks-months (collagens,
  crystallins, myelin proteins).
- Average mammalian-cell protein t1/2 ~ 1-2 days.
- Whole-body protein turnover ~ 250 g / day in
  a 70 kg adult (~ 3-4 % of body protein).
- Most degradation is **UPS** (cytoplasmic +
  nuclear soluble proteins) + **autophagy** (bulk
  + organelle + aggregate).

Protein quality:
- **Net positive nitrogen balance** — growth,
  pregnancy, recovery from injury.
- **Net negative balance** — starvation, sepsis,
  cancer cachexia, immobilisation.
- **Sarcopenia** — age-related muscle loss;
  protein + resistance exercise mitigation.

## Clinical labs

- **ALT, AST** — liver injury (ALT more specific).
- **Albumin** — synthesis indicator (chronic
  liver disease, malnutrition); negative acute-
  phase reactant.
- **Total protein**.
- **NH₃** — hepatic encephalopathy + UCD
  workup.
- **Plasma + urine amino-acid profiles** —
  IEM workup.
- **MCAD acylcarnitine profile** — newborn
  screening.
- **Serum BUN + creatinine** — renal function +
  protein catabolism.
