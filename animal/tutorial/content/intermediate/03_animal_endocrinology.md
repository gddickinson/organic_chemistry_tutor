# Animal endocrinology + hormones

The endocrine system uses circulating hormones to
coordinate metabolism, growth, reproduction, stress
responses, water balance, and behaviour across the
whole organism.  Vertebrate endocrinology has a
well-defined organ + axis structure; invertebrate
endocrinology adds dramatic developmental + ecdysial
control.

## Vertebrate endocrine glands

| Gland | Major hormones | Function |
|-------|----------------|----------|
| Hypothalamus | TRH, CRH, GnRH, GHRH, somatostatin, dopamine, ADH, oxytocin (the latter two via posterior pituitary) | Master regulator |
| Pituitary anterior | TSH, ACTH, FSH, LH, GH, prolactin | Drives downstream glands |
| Pituitary posterior | ADH (vasopressin), oxytocin | Stored + released; made in hypothalamus |
| Pineal | Melatonin | Circadian + sleep |
| Thyroid | T3, T4, calcitonin | Metabolism, calcium |
| Parathyroid | PTH | Calcium homeostasis |
| Adrenal cortex | Cortisol, aldosterone, DHEA | Stress, salt, sex precursor |
| Adrenal medulla | Adrenaline, noradrenaline | Sympathetic / fight-or-flight |
| Pancreas (islets) | Insulin, glucagon, somatostatin, PP | Glucose homeostasis |
| Gonads (ovaries) | Oestrogens, progesterone, inhibin | Female reproduction |
| Gonads (testes) | Testosterone, inhibin | Male reproduction |
| Kidney | EPO, renin, calcitriol | Erythropoiesis, BP, calcium |
| Heart | ANP, BNP | Sodium + water excretion |
| GI tract | Gastrin, secretin, CCK, ghrelin, GLP-1, GIP | Digestion + appetite |
| Adipose | Leptin, adiponectin | Energy balance |

## The major axes

### HPA — Hypothalamic-Pituitary-Adrenal

Stress + glucocorticoid axis.

```
Stressor → hypothalamic PVN → CRH + AVP →
anterior pituitary corticotrophs → ACTH →
adrenal cortex → cortisol → systemic effects +
negative feedback at hypothalamus + pituitary.
```

Cortisol effects:
- Mobilises glucose + amino acids + fatty acids
  for fuel.
- Suppresses immune + inflammatory responses
  (basis of glucocorticoid pharmacology).
- Maintains BP via vascular sensitivity to
  catecholamines.
- Impacts mood + cognition.

Diurnal rhythm — peak in early morning, trough
overnight.

Diseases:
- **Cushing's** — excess cortisol (pituitary tumour =
  Cushing's disease, ectopic ACTH, adrenal tumour,
  iatrogenic steroid use).
- **Addison's** — adrenal insufficiency.
- **Stress-related disorders** — chronic HPA
  dysregulation in depression, PTSD, chronic disease.

### HPT — Hypothalamic-Pituitary-Thyroid

```
Hypothalamus → TRH → pituitary thyrotrophs → TSH →
thyroid follicular cells → T4 + T3 → systemic
metabolism + negative feedback.
```

T3 / T4:
- Increase basal metabolic rate.
- Drive growth + development (especially
  CNS — neonatal hypothyroidism causes severe
  cognitive impairment if not treated).
- Regulate cardiac contractility + heart rate.
- Influence body temperature.

Synthesised from iodine + thyroglobulin in thyroid
follicles; stored in colloid.  TBG (thyroxine-
binding globulin) carries in blood.  T4 → T3
peripheral conversion by deiodinases (D1, D2);
D3 inactivates.

Diseases:
- **Hyperthyroidism** (Graves', toxic adenoma) —
  weight loss, palpitations, anxiety, heat
  intolerance.
- **Hypothyroidism** (Hashimoto's, post-radioiodine,
  iodine deficiency) — fatigue, weight gain, cold
  intolerance, cretinism if congenital + untreated.
- **Endemic goitre** — iodine deficiency; salt
  iodisation programmes.

### HPG — Hypothalamic-Pituitary-Gonadal

```
Hypothalamus → GnRH (pulsatile!) → pituitary
gonadotrophs → FSH + LH → gonads → sex steroids +
gametogenesis + negative + positive feedback.
```

GnRH pulsatility is essential — continuous GnRH
DESENSITISES (basis of GnRH-agonist therapy in
prostate cancer + endometriosis + central
precocious puberty).

Female cycle:
- Follicular phase (FSH-dominant, oestrogen rises).
- Ovulation (LH surge — oestrogen positive feedback).
- Luteal phase (progesterone-dominant).

Male:
- Continuous spermatogenesis (LH → Leydig cells →
  testosterone; FSH → Sertoli cells → spermatogenesis
  support).

Diseases / pharmacology:
- **PCOS** — hyperandrogenism + anovulation.
- **Hypogonadism** — primary (gonadal failure) vs
  secondary (HPG axis).
- **GnRH agonists** (leuprolide, goserelin,
  triptorelin) + **antagonists** (cetrorelix,
  ganirelix, degarelix).
- **Selective oestrogen-receptor modulators (SERMs)**
  — tamoxifen, raloxifene, ospemifene.
- **Aromatase inhibitors** — letrozole, anastrozole,
  exemestane.
- **Anti-androgens** — bicalutamide, enzalutamide,
  apalutamide.

### Renin-Angiotensin-Aldosterone System (RAAS)

```
↓ BP / ↓ Na → renal renin → angiotensinogen → Ang I
→ ACE → Ang II (vasoconstriction + aldosterone
release + thirst + ADH release) → aldosterone
→ Na reabsorption + K excretion in DCT/CD →
↑ BP / ↑ blood volume.
```

Drug targets:
- ACEi (-pril) blocks ACE.
- ARBs (-sartan) block AT1 receptor.
- Aldosterone antagonists (spironolactone,
  eplerenone).
- Renin inhibitors (aliskiren).
- Neprilysin inhibitor + ARB (sacubitril/valsartan)
  for heart failure.

## Glucose homeostasis

Insulin (β-cells) + glucagon (α-cells) +
somatostatin (δ-cells) + PP (PP cells) +
incretins (GLP-1, GIP from GI L + K cells).

### Insulin

- Released in response to glucose, amino acids,
  vagal stimulation, incretins.
- Promotes:
  - Glucose uptake (GLUT4 translocation in muscle +
    adipose).
  - Glycogen synthesis (liver + muscle).
  - Lipogenesis + lipid storage.
  - Protein synthesis.
  - K+ uptake.
- INHIBITS lipolysis + gluconeogenesis +
  glycogenolysis.

### Glucagon

- Released in response to LOW glucose, amino acids,
  catecholamines.
- Promotes:
  - Glycogenolysis.
  - Gluconeogenesis.
  - Lipolysis.
- Antagonises insulin.

### Diabetes

- **Type 1** — autoimmune β-cell destruction;
  insulin-dependent.
- **Type 2** — insulin resistance + relative
  β-cell failure; lifestyle + genetic.
- **MODY** — monogenic; HNF1A, HNF4A, GCK most
  common.
- **Gestational** — pregnancy-onset.

Drug pharmacology revolution since ~ 2010:
- **Metformin** — first-line; AMPK + microbiome
  + liver gluconeogenesis suppression.
- **Sulfonylureas** (glibenclamide, gliclazide) —
  KATP-channel closure → β-cell insulin release.
- **DPP-4 inhibitors** (sitagliptin et al.) — boost
  endogenous incretin.
- **GLP-1 receptor agonists** (semaglutide,
  liraglutide, dulaglutide, tirzepatide — also GIP
  agonist) — multiple beneficial effects on glucose
  + weight + CV + renal outcomes.
- **SGLT2 inhibitors** (empagliflozin, dapagliflozin,
  canagliflozin) — glucosuria + CV + renal benefits;
  increasingly used in heart failure + CKD even
  without diabetes.
- **Insulin** — multiple analogue formulations
  (rapid: lispro / aspart / glulisine; long: glargine
  / detemir / degludec; ultra-long: insulin icodec).

## Stress + adrenergic responses

The "fight-or-flight" response:
- Sympathetic nervous system activation →
  noradrenaline release at organs.
- Adrenal medulla → adrenaline + noradrenaline
  systemic burst.
- Effects: ↑ HR + force, ↑ BP, ↑ glucose, ↑ FFA
  mobilisation, bronchodilation, pupil dilation,
  ↓ GI motility + secretion, sphincter contraction,
  glycogenolysis + lipolysis.
- Mediated by α + β adrenergic receptors (PH-1.0 /
  PH-2.0 catalogue covers in detail).

Chronic stress consequences:
- Hypertension, cardiovascular disease.
- Immunosuppression.
- Insulin resistance.
- Hippocampal atrophy + memory impairment.
- Mood disorders.

## Calcium + phosphate homeostasis

Tightly regulated by PTH + calcitriol (active
vitamin D) + calcitonin.

### PTH

- Released in response to LOW Ca²⁺.
- Effects:
  - ↑ bone resorption (RANKL → osteoclast).
  - ↑ renal Ca reabsorption + phosphate excretion.
  - ↑ renal calcitriol synthesis.
- Net: raises serum Ca, lowers serum P.

### Calcitriol (1,25-(OH)2-D3)

- Made in kidney from 25-OH-D3 by 1α-hydroxylase
  (PTH-stimulated).
- Effects:
  - ↑ intestinal Ca + P absorption.
  - ↑ bone mineralisation + remodelling.

### Calcitonin

- Released in response to HIGH Ca²⁺.
- Inhibits osteoclasts; minor role in human
  Ca homeostasis.

## Reproductive endocrinology + behaviour

Sex steroids (testosterone, oestradiol, progesterone,
DHEA) influence:
- Reproductive function + gametogenesis.
- Secondary sexual characters.
- Bone density (oestrogen protective).
- Cardiovascular tone.
- CNS effects on mood, cognition, libido.
- Metabolism + body composition.

Reproductive cycles vary across mammals:
- Continuous breeders (humans, primates, mice).
- Seasonal breeders (deer, sheep, photoperiod-driven).
- Induced ovulators (cats, rabbits, ferrets;
  ovulation triggered by mating).
- Monestrous (single oestrus per year — bears,
  wolves) vs polyestrous (multiple — many rodents).

## Invertebrate endocrinology

A different toolbox dominates invertebrate biology.

### Insect ecdysone + juvenile hormone (JH)

- **20-hydroxyecdysone (20E)** — steroid hormone
  triggering moulting + metamorphosis.
  Ecdysone receptor (EcR) is a nuclear-hormone
  receptor; binds + activates ecdysone-responsive
  gene cascades.
- **Juvenile hormone (JH)** — sesquiterpenoid;
  determines whether each moult preserves larval
  features or promotes metamorphosis.  High JH +
  ecdysone → larval-larval moult; low JH +
  ecdysone → pupation; absent JH + ecdysone in
  pupa → adult.
- **Insect growth regulators (IGRs)** as insecticides
  — methoprene (JH analogue), tebufenozide
  (ecdysone-like).

### Insect neuropeptides

- **Bursicon** — cuticle tanning.
- **Eclosion hormone** — moult + eclosion.
- **PTTH** (prothoracicotropic hormone) — drives
  ecdysone production.
- **Allatostatins / allatotropins** — regulate JH
  production from corpora allata.

### Crustacean hormones

- Y-organ produces ecdysteroids; X-organ / sinus
  gland produces molt-inhibiting hormone.
- Crustacean hyperglycaemic hormone (CHH).
- Vitellogenin + reproductive hormones.

### Cnidarians + flatworms

Smaller hormone repertoires.  Some
peptides + retinoids + steroids documented.
*Hydra* head activator + head inhibitor (Schaller +
Bode 1970s).

### Octopus + cephalopod neurosecretion

Optic-gland neurosecretion drives reproductive
maturation + post-spawning death (semelparous).

## Endocrine disruptors

Anthropogenic chemicals interfering with hormone
systems:
- **DDT + metabolites** — eggshell thinning in
  raptors + persistent in environment.
- **Bisphenols (BPA, BPS, BPF)** — weak oestrogenic.
- **Phthalates** — anti-androgenic.
- **Atrazine** — gonadotropic disruption.
- **PFOS / PFOA / PFAS** — multiple endocrine +
  immune effects; "forever chemicals".
- **Tributyltin** — antifouling; imposex in
  gastropods.
- **Synthetic oestrogens in waste water** — feminisation
  of fish populations.

Wildlife declines + altered sex ratios + behavioural
changes in wild animal populations link to
endocrine-disruptor exposure.
