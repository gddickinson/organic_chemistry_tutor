# Animal immunology essentials

The immune system protects against pathogens +
cancers + dysregulated self while tolerating
commensals + tissues.  Vertebrate immunology is the
clinical paradigm; comparative immunology shows
that adaptive immunity is a vertebrate innovation
elaborated atop a deeply ancient innate system.

## Two arms of vertebrate immunity

### Innate

Hardwired + ancient + first-line + non-specific
(or pattern-specific):
- **Physical + chemical barriers** — skin, mucosae,
  defensins, lysozyme, gastric acid, mucociliary
  clearance.
- **Pattern-recognition receptors (PRRs)** —
  TLR1-10, NOD-like receptors (NLRs), RIG-I-like
  (RLRs), C-type lectins (Dectin-1).  Recognise
  PAMPs (bacterial / viral / fungal molecular
  patterns).
- **Complement system** — > 30 plasma proteins;
  three activation pathways (classical, alternative,
  lectin) → MAC (membrane attack complex) +
  opsonisation + chemotaxis.
- **Phagocytes** — neutrophils, monocytes /
  macrophages, dendritic cells.
- **NK cells** — kill stressed + transformed cells
  via "missing self" + ADCC.
- **Cytokines + chemokines** — coordinate response
  (IL-1, IL-6, IL-8, IL-12, TNF-α, type-I IFN).
- **Inflammation** — acute (vasodilation +
  permeability + neutrophil influx) +
  chronic.

### Adaptive

Slower, antigen-specific, generates memory:
- **B cells** — produce antibodies (immunoglobulins).
- **T cells** — CD4+ helpers, CD8+ cytotoxic,
  Tregs, etc.
- **Antigen presentation** — MHC class I (CD8 T
  recognition; intracellular pathogens) + MHC
  class II (CD4 T recognition; extracellular
  pathogens).
- **Receptor diversity** generated by
  **V(D)J recombination** + **somatic hypermutation**
  + **class switch recombination**.

## Innate immunity in detail

### Pattern-recognition receptors

| Receptor | Location | Ligand |
|----------|----------|--------|
| TLR2 | Cell surface | Lipoteichoic acid, peptidoglycan, lipopeptide |
| TLR3 | Endosomal | dsRNA |
| TLR4 | Cell surface | LPS (lipid A) |
| TLR5 | Cell surface | Flagellin |
| TLR7 / 8 | Endosomal | ssRNA |
| TLR9 | Endosomal | Unmethylated CpG DNA |
| NOD1 / 2 | Cytoplasm | Peptidoglycan fragments |
| NLRP3 | Cytoplasm | Diverse danger signals → inflammasome |
| RIG-I / MDA5 | Cytoplasm | Viral RNA → IFN response |
| cGAS-STING | Cytoplasm | dsDNA → IFN response |

Engagement triggers MyD88 / TRIF / MAVS / STING
adapter complexes → MAPK + NF-κB + IRF activation
→ pro-inflammatory cytokines + type-I IFN.

### Complement

Three pathways:
- **Classical** — C1q binds antibody-bound antigen.
- **Lectin** — MBL / ficolin binds carbohydrate
  patterns.
- **Alternative** — spontaneous C3 hydrolysis;
  amplifies on pathogen surfaces.

All converge on C3 cleavage → C3b (opsonin) + C3a
(anaphylatoxin) → C5 cleavage → C5a (chemotaxin)
+ C5b → MAC (C5b-9; lyses gram-negatives + some
membranes).

Complement deficiencies:
- **C1, C2, C4** — SLE-like syndrome.
- **C3** — recurrent pyogenic infections.
- **MAC components (C5-9)** — recurrent *Neisseria*
  infections.
- **C1 esterase inhibitor** — hereditary angioedema.

### Phagocytes

- **Neutrophils** — first responders; short-lived;
  > 50 % of WBCs in human blood.  Defensins,
  myeloperoxidase, ROS, NETs (neutrophil
  extracellular traps).
- **Monocytes / macrophages** — tissue-resident
  + recruited; long-lived; phagocytosis +
  cytokine production + antigen presentation.
- **Dendritic cells** — bridge innate + adaptive;
  professional antigen presenters; specialised
  subsets (cDC1, cDC2, pDC).

### Inflammasomes

NLRP3 / NLRC4 / AIM2 inflammasomes assemble in
response to PAMPs + DAMPs → caspase-1 activation
→ IL-1β + IL-18 maturation + pyroptosis.  Drives
many sterile + infectious inflammatory states.
NLRP3 is a target of new biologics + small-
molecule inhibitors (CAPS treatment with
canakinumab; gout colchicine; emerging NLRP3
inhibitors).

## Adaptive immunity in detail

### B cells + antibodies

BCR = membrane-bound antibody.  Activation requires
antigen + T-helper signal in most cases.

Antibody isotypes:
- **IgM** — first produced; pentameric; complement
  fixation.
- **IgD** — minor; surface marker.
- **IgG** — most abundant in serum; 4 subclasses;
  crosses placenta; long half-life (~ 21 days);
  opsonisation + ADCC + neutralisation +
  complement.
- **IgA** — mucosal immunity; secreted dimeric.
- **IgE** — mast cell + basophil; allergy + parasite.

Antibody diversity:
- **V(D)J recombination** — RAG1/2 + TdT generate
  combinatorial diversity (heavy chain V × D × J;
  light chain V × J).
- **Junctional diversity** — NHEJ-mediated end
  processing.
- **Somatic hypermutation** — AID-driven point
  mutations in germinal center B cells →
  affinity maturation.
- **Class switch recombination** — AID-mediated
  switch from IgM → IgG / IgA / IgE.

### T cells

TCR recognises peptide bound to MHC.

T-cell subsets:
- **Naive** — never activated.
- **CD4+ helpers** —
  - **Th1** — IFN-γ; intracellular pathogens.
  - **Th2** — IL-4, IL-5, IL-13; helminths + allergy.
  - **Th17** — IL-17; extracellular bacteria + fungi
    + autoimmune disease.
  - **Tfh** — follicular helpers; B-cell help in GC.
  - **Treg** — Foxp3+; immune tolerance.
- **CD8+ cytotoxic** — kill via perforin + granzyme +
  Fas/FasL.
- **γδ T cells** — innate-adaptive bridge; epithelial
  immunity.

### MHC

Class I (HLA-A, B, C in humans) — present cytosolic
peptides; expressed on all nucleated cells; CD8 T
recognition.

Class II (HLA-DR, DP, DQ) — present extracellular
peptides via endocytic processing; expressed on
APCs (DCs, B cells, macrophages); CD4 T recognition.

MHC polymorphism is the most variable in the genome
— drives transplantation rejection + many autoimmune
disease associations + pharmacogenomics
(HLA-B*57:01 + abacavir; HLA-B*15:02 + carbamazepine).

### Memory

- **Memory B cells** — produce high-affinity, class-
  switched antibodies on re-exposure.
- **Memory T cells** — central memory (Tcm) +
  effector memory (Tem) + tissue-resident (Trm).
- **Long-lived plasma cells** in bone marrow
  produce antibodies for years.

Vaccines exploit immunological memory — covered
in **MB-3.0 intermediate "Vaccines" lesson**.

## Mucosal + epithelial immunity

Specialised compartments at body surfaces:
- **GALT (gut-associated)** — Peyer's patches,
  lamina propria lymphocytes, IgA-producing plasma
  cells.
- **NALT** (nasopharyngeal), **BALT** (bronchial),
  **MALT** (mucosa-associated overall).
- **Microbiome-immune** crosstalk regulates
  local + systemic immunity.

## Tolerance + autoimmunity

### Mechanisms of tolerance

- **Central tolerance** — thymic deletion of strong
  self-reactive T cells (AIRE-mediated thymic
  expression of peripheral self-antigens).  B cells
  similarly tested in marrow.
- **Peripheral tolerance** — anergy, deletion, Treg
  suppression.

### Failure → autoimmune disease

- **SLE** — anti-nuclear + anti-dsDNA antibodies;
  multi-system.
- **Type 1 diabetes** — anti-islet T cells.
- **Multiple sclerosis** — autoreactive T cells +
  CNS demyelination.
- **Rheumatoid arthritis** — anti-citrullinated
  protein + RF.
- **Hashimoto's** + **Graves'** — thyroid auto-
  antibodies.
- **Crohn's + UC** — inflammatory bowel disease.
- **Psoriasis** + **psoriatic arthritis** — IL-17 /
  IL-23 axis.

## Allergy + hypersensitivity

Gell + Coombs classification:
- **Type I (immediate)** — IgE-mediated mast cell +
  basophil degranulation.  Anaphylaxis, allergic
  rhinitis, asthma, food allergies.
- **Type II (cytotoxic)** — antibody binds cell
  surface antigen → ADCC + complement.  HDN, AIHA,
  pemphigus.
- **Type III (immune complex)** — antigen-antibody
  complexes deposit in tissue → complement +
  inflammation.  SLE nephritis, post-strep
  glomerulonephritis, serum sickness.
- **Type IV (delayed, T-cell-mediated)** — contact
  dermatitis, TB skin test, transplant rejection.

## Cancer immunology

The immune system surveils + eliminates emerging
tumours; tumours that escape do so via:
- Loss of MHC class I → invisible to CD8.
- Recruitment of Tregs + MDSCs.
- Expression of checkpoint ligands (PD-L1, B7
  family) → T-cell exhaustion.

**Checkpoint inhibitors** — pembrolizumab + nivolumab
(anti-PD-1) + atezolizumab + durvalumab (anti-PD-L1)
+ ipilimumab (anti-CTLA-4) — release the brakes;
revolutionised oncology since 2014; James Allison +
Tasuku Honjo Nobel 2018.

**CAR-T cells** — autologous T cells engineered with
chimeric antigen receptors targeting tumour antigens
(CD19 in B-cell malignancies, BCMA in MM).
Covered in **PH-3.0 graduate "Modern modalities"**.

## Comparative immunology — across phyla

### Invertebrates

- **No adaptive immunity** in classical sense.
- **Innate immunity** richer than once thought:
  - PRR systems (Toll first found in *Drosophila*!).
  - Antimicrobial peptides (cecropins, defensins,
    drosomycin, etc.) — diverse + abundant.
  - Phenoloxidase cascade — melanisation +
    encapsulation of pathogens.
  - Phagocytic haemocytes.
- **Some adaptive-like memory** — *Drosophila* +
  *Anopheles* show specific immune priming;
  mechanisms unclear but RNAi-mediated antiviral
  immunity has memory-like properties.

### Jawless vertebrates (lamprey + hagfish)

- Have ADAPTIVE IMMUNITY — but with a TOTALLY
  DIFFERENT receptor system.
- **Variable lymphocyte receptors (VLRs)** built
  from leucine-rich repeats (LRRs) instead of
  immunoglobulin domains.
- Convergent evolution of adaptive immunity.

### Jawed vertebrates

- All share the canonical Ig + TCR + MHC + RAG-
  recombinase adaptive system.
- Originated ~ 500 Ma with gnathostomes.

### Plants

- No mobile immune cells; cell-autonomous immunity
  + R-genes + RNA silencing.  Covered in **BT-3.0
  intermediate "Plant defence" lesson**.
