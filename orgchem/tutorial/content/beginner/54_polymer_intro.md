# Polymer chemistry beginner intro

A **polymer** is a large molecule made of repeating units
(monomers). The polymer industry produces ~ 400 million
tonnes / yr — plastics, rubbers, fibres, adhesives,
coatings, biopolymers.

## Polymerisation types

### Chain-growth polymerisation

A reactive species (radical, cation, anion, organometallic)
adds monomer one at a time. Chain length grows in seconds:

- **Free-radical**: vinyl monomers (styrene → polystyrene,
  ethylene → LDPE, MMA → PMMA, butadiene → polybutadiene).
- **Cationic**: isobutylene → butyl rubber.
- **Anionic**: styrene + n-BuLi → living polystyrene.
- **Coordination (Ziegler-Natta + metallocene)**: ethylene
  → HDPE; propylene → isotactic PP.

### Step-growth (condensation) polymerisation

Bifunctional monomers react gradually; chain length grows
slowly + only at high conversion (> 99 %):

- **Polyamide (nylon)**: diamine + diacid → -NH-CO- repeats.
- **Polyester (PET, PLA)**: diol + diacid (or hydroxyl-acid).
- **Polycarbonate**: BPA + phosgene.
- **Polyurethane**: diisocyanate + diol.

## Common polymers

| Polymer | Monomer | Type | Use |
|---------|---------|------|-----|
| PE (polyethylene) | ethylene | radical / Z-N | bottles, bags |
| PP (polypropylene) | propylene | Z-N | car parts, fibres |
| PVC | vinyl chloride | radical | pipes |
| PS (polystyrene) | styrene | radical | packaging |
| PMMA | methyl methacrylate | radical | acrylic glass |
| PTFE (Teflon) | tetrafluoroethylene | radical | non-stick |
| PET | terephthalic acid + EG | step | bottles, fibres |
| Nylon 6,6 | hexamethylene-diamine + adipic acid | step | textiles |
| Nylon 6 | caprolactam | ROP | textiles |
| PC (Lexan) | BPA + phosgene | step | helmets |
| PU | diisocyanate + diol | step | foams |
| PLA | lactic acid (lactide) | ROP | bioplastic |
| Natural rubber | isoprene | natural | tyres |

## Tacticity

Stereochemistry of repeat units along the chain:

- **Isotactic** — all stereocentres same → crystalline,
  high-melting (isotactic PP, T_m 165 °C).
- **Syndiotactic** — alternating stereocentres → still
  semi-crystalline.
- **Atactic** — random → amorphous, soft (atactic PP, no
  T_m).

Ziegler-Natta + metallocene catalysts control tacticity →
billions of dollars of industrial value.

## Glass transition + melting

- **T_g (glass transition)**: temperature where amorphous
  polymer goes from glassy (rigid) to rubbery. Depends on
  chain flexibility.
- **T_m (melting point)**: where crystalline regions melt.
  T_g < T_m typically.

Examples:

| Polymer | T_g | T_m |
|---------|-----|-----|
| PMMA | 105 °C | (atactic) |
| PS | 100 °C | (atactic) |
| PE (HDPE) | -120 °C | 130 °C |
| PP (isotactic) | -10 °C | 165 °C |
| PET | 70 °C | 260 °C |
