# Common reagent abbreviations + what they mean

Synthesis schemes are full of acronyms — LDA, DIBAL, mCPBA,
DMP. Once you've memorised the workhorses you can read most
papers fluently. Here's the canonical short-list.

## Bases

| Abbrev | Full name | Use |
|--------|-----------|-----|
| **NaH** | Sodium hydride | Strong + irreversible; deprotonates alcohols, amides, malonates |
| **NaOH / KOH** | Sodium / potassium hydroxide | Aqueous; saponifies esters, deprotonates phenols |
| **LDA** | Lithium diisopropylamide | Bulky, very strong (pKaH 36); kinetic enolates |
| **LiHMDS** | Lithium bis(trimethylsilyl)amide | Even bulkier; selective deprotonations |
| **NaHMDS / KHMDS** | Sodium / potassium variants | Less basic but more soluble in non-polar solvents |
| **DBU** | 1,8-diazabicyclo[5.4.0]undec-7-ene | Hindered amidine; non-nucleophilic, mild |
| **DBN** | 1,5-diazabicyclo[4.3.0]non-5-ene | Smaller cousin of DBU |
| **TEA / Et₃N** | Triethylamine | Mild proton scavenger; good for acid-chloride chemistry |
| **DIPEA / Hünig's base** | N,N-diisopropylethylamine | Bulky non-nucleophilic; cleaner than TEA |
| **DMAP** | 4-dimethylaminopyridine | Acyl-transfer catalyst (5-10 mol %) |
| **t-BuOK** | Potassium tert-butoxide | Strong + bulky; E2 eliminations toward Hofmann |
| **n-BuLi / s-BuLi / t-BuLi** | n-, s-, t-butyllithium | Very strong; metallate aromatic / vinylic C-H |

## Reducing agents

| Abbrev | Full name | Use |
|--------|-----------|-----|
| **NaBH₄** | Sodium borohydride | Aldehyde / ketone → alcohol; tolerates esters |
| **LiAlH₄ / LAH** | Lithium aluminium hydride | Reduces almost everything (esters, amides, nitriles) |
| **DIBAL / DIBAL-H** | Diisobutylaluminium hydride | Stops esters at the aldehyde stage at -78 °C |
| **L-Selectride** | Lithium tri-sec-butylborohydride | Bulky, stereoselective ketone reduction |
| **NaBH(OAc)₃ / STAB** | Sodium triacetoxyborohydride | Reductive amination |
| **NaBH₃CN** | Sodium cyanoborohydride | Mild reductive amination at pH 6-8 |
| **H₂ / Pd-C** | Hydrogen + palladium on carbon | Alkene + alkyne hydrogenation; benzyl debenzylation |
| **H₂ / Lindlar** | H₂ over Pd / Pb / quinoline | Alkyne → cis alkene only (no over-reduction) |
| **Na / NH₃ (Birch)** | Sodium in liquid ammonia | Aromatic ring → 1,4-cyclohexadiene; alkyne → trans alkene |
| **Sm I₂** | Samarium(II) iodide (Kagan's reagent) | Single-electron reductant, pinacol coupling |

## Oxidising agents

| Abbrev | Full name | Use |
|--------|-----------|-----|
| **PCC** | Pyridinium chlorochromate | 1° alcohol → aldehyde (stops there) |
| **PDC** | Pyridinium dichromate | Similar to PCC, milder |
| **CrO₃ / H₂SO₄ / H₂O (Jones)** | Jones reagent | 1° alcohol → COOH; 2° alcohol → ketone |
| **DMP** | Dess-Martin periodinane | Mild + room-T alcohol oxidation; replaced PCC in modern labs |
| **Swern** | (COCl)₂ / DMSO / Et₃N | Mild alcohol oxidation; smelly Me₂S byproduct |
| **TPAP / NMO** | Tetrapropylammonium perruthenate | Catalytic oxidation of alcohols |
| **mCPBA** | meta-chloroperoxybenzoic acid | Epoxidation of alkenes; Baeyer-Villiger oxidation |
| **OsO₄** | Osmium tetroxide | Syn dihydroxylation of alkenes (cis-diol) |
| **KMnO₄** | Potassium permanganate | Strong oxidant; cleaves alkenes to acids |
| **O₃** | Ozone | Cleaves alkenes; Zn/AcOH workup → aldehyde, H₂O₂ → COOH |
| **NaIO₄** | Sodium periodate | Cleaves diols to two C=O |
| **DDQ** | 2,3-dichloro-5,6-dicyano-1,4-benzoquinone | Aromatisation; PMB-ether deprotection |

## Halogenation + activating agents

| Abbrev | Full name | Use |
|--------|-----------|-----|
| **NBS** | N-bromosuccinimide | Allylic / benzylic bromination; bromohydrin formation |
| **NCS** | N-chlorosuccinimide | Mild source of Cl⁺ |
| **NIS** | N-iodosuccinimide | Mild source of I⁺ |
| **TsCl** | p-toluenesulfonyl chloride | Activate OH as a tosylate (good LG) |
| **MsCl** | Methanesulfonyl chloride | Activate OH as a mesylate |
| **TfOH / Tf₂O** | Triflic acid / anhydride | Super-strong acid; activate OH as triflate (best LG) |
| **SOCl₂** | Thionyl chloride | OH → Cl, COOH → COCl |
| **PBr₃** | Phosphorus tribromide | OH → Br |
| **DAST** | Diethylaminosulfur trifluoride | OH → F, C=O → CF₂ |

## Coupling + amide-bond reagents

| Abbrev | Full name | Use |
|--------|-----------|-----|
| **DCC** | N,N'-dicyclohexylcarbodiimide | Carboxylic acid activator; ester / amide formation |
| **EDC / EDC-HCl** | 1-ethyl-3-(3-dimethylaminopropyl)carbodiimide HCl | Water-soluble DCC, protein conjugation |
| **HATU, HBTU, PyBOP** | Uronium / phosphonium salts | Modern peptide-coupling reagents |
| **CDI** | Carbonyldiimidazole | Mild carboxylic-acid activator |
| **T3P** | Propanephosphonic acid anhydride | Scalable amide coupling, low racemisation |

## Protecting groups (also their reagent abbreviations)

| Abbrev | Full name | Protects |
|--------|-----------|----------|
| **Boc / Boc₂O** | tert-butyloxycarbonyl | Amine — TFA labile |
| **Fmoc / Fmoc-Cl** | 9-fluorenylmethyloxycarbonyl | Amine — base labile (piperidine) |
| **Cbz / Z** | Benzyloxycarbonyl | Amine — H₂/Pd labile |
| **Bn / BnBr** | Benzyl | OH, NH — H₂/Pd labile |
| **Bz / BzCl** | Benzoyl | OH — base labile |
| **Ac / Ac₂O** | Acetyl | OH, NH — base labile |
| **TBS / TBDMS** | tert-butyldimethylsilyl | OH — F⁻ (TBAF) labile |
| **TMS / TMS-Cl** | Trimethylsilyl | Most labile silyl — quick protection |
| **TBDPS** | tert-butyldiphenylsilyl | Most stable silyl |
| **THP / DHP** | Tetrahydropyranyl | OH — acid labile |
| **MOM / MOMCl** | Methoxymethyl | OH — acid labile |
| **PMB / PMBCl** | p-methoxybenzyl | OH — DDQ labile |
