# Acids and bases — Brønsted vs Lewis

Two complementary definitions of "acid" + "base" run side-by-
side in organic chemistry. Knowing which framework applies in a
given context is the difference between predicting + guessing.

## Brønsted-Lowry: proton transfer

A **Brønsted acid** donates a proton (H⁺); a **Brønsted base**
accepts one. The classic case:

```
HCl  +  H₂O   →   H₃O⁺  +  Cl⁻
acid    base      conj.    conj.
                  acid     base
```

The pKa scale ranks Brønsted acids by their tendency to
release a proton — lower pKa = stronger acid. See lesson 32 for
the full conjugate-pair framework.

## Lewis: electron-pair transfer

A **Lewis acid** accepts an electron pair (it has an empty or
partially-empty orbital); a **Lewis base** donates one (it has
a lone pair).

```
BF₃  +  :NH₃   →   F₃B–NH₃
Lewis   Lewis      adduct
acid    base
```

Every Brønsted acid is also a Lewis acid (H⁺ accepts an
electron pair to form a bond). But not every Lewis acid is a
Brønsted acid — BF₃, AlCl₃, Mg²⁺, Cu²⁺ are Lewis acids with no
proton to donate.

## Where each framework dominates

| Framework | Domain |
|-----------|--------|
| Brønsted | Aqueous solution, biology, acid-base titrations, pKa |
| Lewis | Friedel-Crafts (AlCl₃ activation), Grignard chemistry, transition-metal catalysis, organocatalysis |

## Common Lewis acids in organic synthesis

- **AlCl₃** — Friedel-Crafts alkylation + acylation.
- **BF₃·OEt₂** — Mukaiyama aldol activator.
- **TiCl₄** — Lewis-acid catalyst for many cycloadditions.
- **Sc(OTf)₃** — water-tolerant Lewis acid.
- **Cu(II)** — chelating Lewis-acid catalyst for asymmetric
  reactions.
