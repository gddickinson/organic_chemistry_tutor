# Mole fraction + Raoult's law

For mixtures of liquids, the partial pressure of each
component above the solution depends on **mole fraction** —
not mass fraction or volume fraction.

## Mole fraction

For a binary mixture A + B:

```
x_A = n_A / (n_A + n_B)
x_B = n_B / (n_A + n_B)
x_A + x_B = 1
```

Mole fraction is always 0 to 1 + adds to 1 for all
components. Independent of T (unlike molarity).

## Raoult's law (ideal mixtures)

```
P_A = x_A × P_A°            P_B = x_B × P_B°
P_total = P_A + P_B
```

P_A° is the vapour pressure of **pure A** at the given
temperature. P_A is its partial pressure above the
solution.

For ideal mixtures (similar molecules, e.g. benzene +
toluene), this works well.

## Distillation predictions

The vapour above an ideal solution is enriched in the
more volatile component. Mole fraction in vapour:

```
y_A = P_A / P_total = x_A P_A° / P_total
```

If P_A° > P_B° → y_A > x_A. Recondense → 2nd fraction
even more enriched. Repeat → fractional distillation
separates the mixture.

## Deviations from Raoult's law

Real mixtures deviate when intermolecular forces between A
+ B differ from A-A or B-B:

- **Negative deviation** (P_total < ideal) — A-B forces
  *stronger* than A-A or B-B. Acetone + chloroform (H-bond
  to chloroform's CH).
- **Positive deviation** (P_total > ideal) — A-B forces
  *weaker*. Ethanol + benzene (broken H-bonds in EtOH).

Strong positive deviation → **azeotrope** (constant-boiling
mixture) → can't separate by simple distillation.

Famous azeotropes:

- EtOH/H₂O: ~ 96 % EtOH, bp 78.2 °C → max-boiling.
- HCl/H₂O: 20.2 % HCl, bp 108.6 °C → max-boiling.
- THF/H₂O: ~ 95 % THF, max-boiling.

To get past azeotrope: add a third component (benzene,
cyclohexane) — Drier (3 Å sieves) for absolute EtOH; or
azeotropic distillation.

## Henry's law

For dilute gas in liquid:

```
P_gas = k_H × x_gas
```

k_H is the Henry's law constant. CO₂ in soda obeys this
(open the bottle → P_CO₂ drops → CO₂ comes out of
solution).
