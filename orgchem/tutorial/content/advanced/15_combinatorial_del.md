# Combinatorial chemistry + DNA-encoded libraries (DEL)

A combinatorial library is a set of compounds generated by
varying building blocks at multiple positions of a common
scaffold. The promise: one campaign generates millions to
billions of compounds for screening. The reality has shifted
from 1990s "make + test plate-by-plate" to modern **DNA-
encoded libraries** that screen 10⁹-10¹² compounds in a single
tube.

## 1990s: split-and-pool combinatorial chemistry

**Furka, Lam, Houghten** (1991) — split a pool of resin beads
into N portions, react each portion with a different building
block, recombine, then split again for the next position.

After three split-pool cycles with 100 building blocks per
position you have **10⁶ unique compounds**, **one compound
per bead**.

Two readout problems made it tough:

1. **Identity** — how do you know what's on the bead that
   hits in your screen?
2. **Quantity** — one bead has ~ 100 pmol, often not enough
   for follow-up.

Workarounds: encoded tags (orthogonal halogen tags read by
mass spec; sequenced peptide tags), iterative deconvolution
(re-synthesise + bisect winning sub-pools).

## 2000s: parallel synthesis ≠ combinatorial

The hype died. Modern "parallel" pharma chemistry (96- /
384-well plates) with one compound per well replaced split-
pool. Easier identity tracking + better characterisation,
but smaller library size (~ 10⁴ compounds).

## 2010s: DNA-encoded libraries (DEL) — the renaissance

**Brenner + Lerner** (1992) had the original idea — attach a
DNA tag during each combinatorial step, the cumulative tag
*encodes* the synthetic history. **Praecis Pharmaceuticals**
(later GSK Boston) commercialised it ~ 2008.

### Library construction

1. Start with a DNA-tagged headpiece (e.g., a primary amine).
2. Split into N wells; each well gets a different building
   block + a unique DNA tag ligated.
3. Pool, split, repeat for cycles 2 + 3 + (4).
4. Final library: 10⁹-10¹² unique compounds, each
   covalently linked to a DNA "barcode" that records its
   composition.

### Affinity screening

1. Mix library with immobilised target protein.
2. Wash off non-binders.
3. **PCR-amplify** the DNA still bound to the protein.
4. **Next-gen sequencing** decodes which barcodes survived.
5. Re-synthesise top hits off-DNA + assay traditionally.

The whole campaign uses ~ 100 nM target protein + ~ 1 µL
library — **single-day, single-tube** screening of 10⁹
compounds.

### What can you make?

Reactions must be DNA-compatible: aqueous-tolerant, mild,
non-oxidative.

- **Amide bond formation** — workhorse, > 60 % of DELs.
- **Reductive amination** — NaBH₃CN-tolerant.
- **CuAAC click** — Cu doesn't damage DNA at low load.
- **Suzuki cross-coupling** — gentle Pd / TPPTS in water.
- **Buchwald-Hartwig** — modern DNA-tolerant ligands exist.
- **Photoredox** — emerging on DEL platforms.

## Industrial DELs

- **GSK** — > 100 billion-member libraries; produced soluble-
  guanylate-cyclase activator, several PROTAC starting points.
- **Nuevolution / X-Chem / HitGen / WuXi AppTec** — DEL CRO
  services; Pfizer + Roche + AstraZeneca routine clients.
- **DiCE Therapeutics** — DEL hits for previously
  undruggable targets (Mcl-1, β-catenin / TCF, KRAS-G12C
  early hits).

## Strengths + limits

**Strengths**:

- Dwarfs traditional HTS (10⁶ → 10¹⁰ compounds).
- Cheap per-compound screening (NGS $0.10 per million reads).
- Targets *novel chemical space* not in vendor catalogues.

**Limits**:

- Hits are obligatorily on-DNA — must re-synthesise off-
  DNA + reconfirm activity.
- DEL hits often have suboptimal physicochemistry (high MW,
  high logP) — lots of medicinal-chemistry optimisation
  needed.
- Some target classes (membrane proteins, intrinsically
  disordered) under-perform in DEL.

## Modern combinatorial spaces

- **Make-on-demand virtual libraries** — Enamine REAL space
  (~ 6 billion synthesisable molecules) + AI search +
  on-demand purchase has supplanted physical combinatorial
  libraries for many use cases.
- **Diversity-oriented synthesis (DOS)** — Schreiber's
  alternative: ~ 10³-10⁴ compounds spanning maximally diverse
  scaffolds, deeper biology testing per scaffold.
