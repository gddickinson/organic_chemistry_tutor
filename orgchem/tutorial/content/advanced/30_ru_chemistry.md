# Ru chemistry intro

Ru is the lightest of the **platinum-group metals**
(Ru, Rh, Pd, Os, Ir, Pt). Used in metathesis (Grubbs) +
asymmetric hydrogenation (Noyori) + several other niches.

## Oxidation states

Ru spans 0 to +8. The most common in synthesis:

- Ru(0): Ru₃(CO)₁₂, Ru-naphthalene, Ru-COD complexes;
  catalyst precursors.
- Ru(II): RuCl₂(PPh₃)₃; Grubbs G1, G2, HG2; Noyori
  hydrogenation precatalysts.
- Ru(III): RuCl₃·xH₂O (the most common starting reagent;
  sold as wet brown salt).
- Ru(VI/VIII): RuO₂, RuO₄ — strong oxidants.

## Famous applications

### Olefin metathesis (Grubbs)

Covered in advanced lesson 10. Grubbs' Ru-alkylidene
catalysts (G1, G2, HG2) are bench-stable + tolerant of
many functional groups → workhorse for ring-closing /
cross-metathesis.

### Asymmetric hydrogenation (Noyori)

```
β-ketoester + H₂ + Ru-(BINAP) → β-hydroxy ester + > 99 % ee
```

Ru-BINAP for **functionalised** ketones; uses the directing
group to coordinate Ru. Industrial L-menthol synthesis at
Takasago (~ 100 t/yr).

```
aryl ketone + iPrOH + Ru-(BINAP)(diamine) → chiral alcohol
```

Noyori's bifunctional catalyst — H⁺ from N-H + H⁻ from Ru-
H deliver simultaneously to a "non-functionalised" ketone.

### Transfer hydrogenation

Use iPrOH or HCOOH/Et₃N as the H source (no H₂ gas
needed):

```
RR'C=O + iPrOH + Ru-TsDPEN → R-CHR'-OH + acetone
                       (chiral)            (acetone formed by H transfer)
```

Noyori's Ru/TsDPEN is the workhorse. Also Wills' tethered
variants.

### RuO₄ + RuO₂ — strong oxidants

```
Cleaves alkenes:    RCH=CHR' + RuO₄ → 2 × RCOOH (like KMnO₄)
                                  or RCHO with mild conditions
Cleaves arenes:     ArCH₃ → ArCOOH
                  + with NaIO₄ as co-oxidant
```

Lemieux-von Rudloff variant: catalytic RuO₄ + NaIO₄ →
cleaves alkenes mildly. Replaces ozonolysis when O₃ is
inconvenient.

### Hydroaminomethylation

Ru-catalysed tandem hydroformylation + reductive amination
in one pot (Beller):

```
RCH=CH₂ + CO + H₂ + R'₂NH + Ru cat. → RCH₂CH₂CH₂-NR'₂
```

Industrial-scale production of amines from cheap olefins.

## Ru in C-H activation

- **Ru(II) + N-directing group** (pyridine, oxime, amide)
  → ortho C-H activation. Cheaper alternative to Pd
  C-H functionalisation.
- **Ru-photoredox** — [Ru(bpy)₃]²⁺ — workhorse photocatalyst
  for visible-light single-electron-transfer chemistry
  (covered in lesson 11).

## Ru complexes in OLEDs + dyes

[Ru(bpy)₃]²⁺ + analogues are emissive triplet sensitisers,
used in:

- Photoredox catalysis (lesson 11).
- Dye-sensitised solar cells (DSSCs; Grätzel).
- Phosphorescent OLED dopants.

## Ru in biology

Ru(III) prodrugs explored for cancer treatment:

- **NAMI-A** — anti-metastatic.
- **KP1019** — anti-tumour; binds DNA differently from
  Pt drugs.

Several in clinical trials but none yet approved.
