# Cu chemistry — Ullmann, Glaser, Chan-Lam, CuAAC

Copper is the cheapest transition metal in the cross-
coupling toolbox. Several classical Cu reactions remain
workhorses + new ones (CuAAC, Chan-Lam) have driven
modern bioconjugation.

## Ullmann coupling (1901)

```
2 ArI + Cu (powder) + heat → Ar-Ar
```

The original cross-coupling — ArX + ArX → biaryl with Cu
metal as catalyst, > 200 °C. Modern version uses milder
conditions:

```
ArI + ArOH or ArNH₂ + Cu/I⁻ + DMG ligand at 80-120 °C →
                         Ar-O-Ar' or Ar-NHAr' (biaryl ether or amine)
```

Modified Ullmann: Buchwald + Ma + Taillefer ligand systems
(diamine, β-amino acid, oxalamide) make this work at 80
°C with catalytic Cu.

Used widely for:

- Diaryl ether synthesis.
- Aryl-amine bonds (cheaper alternative to Pd Buchwald-
  Hartwig).
- C-S, C-O, C-N bond formation.

## Glaser coupling (1869) + Hay coupling

```
2 RC≡CH + Cu(I) + O₂ → RC≡C-C≡C-R (diyne)
```

Oxidative homo-coupling of terminal alkynes. The Hay
modification adds TMEDA to make Cu more soluble +
selective.

Use cases:

- **Sondheimer annulenes** — large conjugated rings via
  Glaser oxidative coupling.
- **Conjugated polymer synthesis** — polydiacetylenes.

## Chan-Lam coupling (1998)

```
ArB(OH)₂ + R-NH₂ (or R-OH) + Cu(II) + air → Ar-NHR (or Ar-OR)
```

Cu(II) acetate + boronic acid + amine in air → C-N or C-O
bond at room temperature. **No base, no Pd, no high T**.

Pros:

- Mildest aryl-amination available.
- Air is the oxidant.
- Tolerates acid-sensitive groups.

Cons:

- Slow (12-48 h).
- Boronic-acid-only nucleophile (sometimes a limit).
- Variable yields with hindered amines.

## CuAAC click chemistry (Sharpless / Meldal, 2002)

```
R-N₃ + R'-C≡CH + Cu(I) + ascorbate or amine ligand → 1,4-triazole
```

The bioorthogonal-chemistry workhorse. Mentioned in many
prior lessons (advanced lesson 12 covers it deeply).

## Chiral Cu catalysis

### Asymmetric conjugate addition (1,4-addition)

```
α,β-unsat carbonyl + R₂Zn (or RMgBr) + Cu / chiral ligand
                  → β-substituted carbonyl with > 90 % ee
```

Hoveyda + Feringa pioneered chiral Cu/phosphine systems
for ACA. Industrial applications in pharma SAR.

### Asymmetric hydrogenation

Cu-catalysed asymmetric H₂ less common than Pd / Rh but
emerging — particularly for ketones + imines that don't
work under standard Noyori conditions.

### Asymmetric Henry / Mukaiyama / Mannich

Cu / chiral diamine / chiral pybox catalysts.

## Cu in C-H activation

Cu(I) / Cu(III) cycles enable C-H halogenation,
amination, oxygenation:

- **Daugulis 8-aminoquinoline-directed Cu-catalysed C-H
  functionalisation** (2005) — directed installation of
  C-Br, C-CN, C-NHR.
- **Sanford-style Cu / oxidant** — picolinamide directed
  C-H acetoxylation.

## Cu corrosion inhibition + redox

Cu redox: Cu(0) → Cu(I) → Cu(II) → Cu(III) → all
accessible:

- Cu(II) / Cu(I) — e⁻ transfer key in Glaser, Chan-Lam,
  Wacker.
- Cu(I) / Cu(III) — radical relay; C-H activation; CuAAC.
- Cu(0) — surface chemistry, Ullmann.

## Cu in industry

- **Wacker process** — PdCl₂ + CuCl₂ for ethylene → acetaldehyde
  (~ 200 kt/yr). Cu is the regenerator (CuCl₂ → CuCl).
- **Acetic acid via Cativa** — Ir + I⁻; Cu absent there
  but earlier Monsanto used Co/Cu mixtures.
- **DEL synthesis** — CuAAC is one of the dominant DNA-
  compatible reactions.

## Toxicity + handling

- Cu(II) salts are mild irritants; environmentally OK at
  low levels (essential trace nutrient).
- Cu(I) acetate + Cu(I) iodide — bench-stable; mild.
- Cu(II) triflate — moisture-sensitive; air-stable.
- Cu(NHC) catalysts — air-stable, off-the-shelf.
