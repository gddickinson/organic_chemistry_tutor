# Enamine + iminium catalysis — List + MacMillan deeper

The asymmetric organocatalysis lesson covered the basics
of the 2021 Nobel chemistry. This lesson goes deeper into
how enamine catalysis (List) + iminium catalysis (MacMillan)
work mechanistically + their key reaction classes.

## Enamine catalysis (List, 2000)

### The catalyst

Secondary amine (proline, MacMillan imidazolidinone,
Jørgensen-Hayashi diaryl prolinol silyl ether) + an
α-CH carbonyl substrate:

```
R-CH₂-CHO + R'₂NH (chiral) → R-CH=NR'₂⁺ → R-CH=CR'-NR'₂ (enamine)
                                  ↓
                    + electrophile attacks at α-C → α-substituted carbonyl
                                  ↓
                    + H₂O hydrolysis → final product + free amine
```

### Famous reactions

- **Aldol** (the original List 2000 paper). Proline (10-30
  mol %) + acetone + benzaldehyde → β-hydroxy ketone with
  > 90 % ee.
- **Mannich** with imines.
- **Michael** with nitroalkenes (Hayashi).
- **α-Halogenation** (List, MacMillan).
- **α-Amination** (Cordova, List).
- **α-Oxygenation** with nitrosobenzene → α-oxyaldehyde.
- **Cascade annulation** (List, Hong).

### Why proline works

Proline is a bifunctional catalyst:

- Secondary amine forms enamine.
- COOH H-bonds to + activates the electrophile.
- 5-membered ring + carboxylate fixes the orientation.

This "Houk-List model" predicts the absolute ee in many
cases.

### Beyond proline

- **MacMillan imidazolidinone** — better selectivity for
  some classes (Diels-Alder, 1,4-addition).
- **Jørgensen-Hayashi diaryl prolinol silyl ether** — bulky;
  excellent for Michael, α-funct.
- **Tetrazole proline** (List's analogue) — pKa-shifted
  carboxylate equivalent; works in water.

## Iminium catalysis (MacMillan, 2000)

### The catalyst

Same secondary amine (MacMillan imidazolidinone), but
condenses with α,β-unsaturated aldehyde:

```
R-CH=CH-CHO + R'₂NH (chiral) → R-CH=CH-CH=NR'₂⁺ (iminium)
                                  ↓
                               β-attack by Nu (Michael)
                               or [4+2] with diene (Diels-Alder)
                                  ↓
                         + H₂O → final product + free amine
```

The iminium has a much LOWER LUMO than the parent enal →
electrophilic at β-C.

### Famous reactions

- **Diels-Alder of α,β-unsat aldehydes** (MacMillan 2000)
  → > 90 % ee for endo cycloadducts.
- **Friedel-Crafts of indoles + pyrroles** with
  α,β-unsat aldehydes.
- **Asymmetric epoxidation** of α,β-unsat aldehydes.
- **Asymmetric Mukaiyama-Michael**.
- **Asymmetric organocatalytic photoredox** (MacMillan,
  2008 onwards) — hybrid of enamine + photoredox.

### MacMillan photoredox-organocatalysis hybrid

Combine iminium / enamine catalysis with photoredox:

```
α-amino radical + iminium (chiral cat) → enantioselective coupling
```

Used in α-arylation, α-acylation, decarboxylative coupling
of α-amino acids.

## SOMO catalysis (MacMillan)

A third mode: oxidise the enamine by 1 e⁻ → α-radical
cation that's both electrophilic + chiral:

```
enamine (from chiral amine) + oxidant → α-radical cation
                + nucleophile → α-functionalised product
```

Lets you do reactions that neither enamine nor iminium
catalysis alone can: α-allylation with allyl silanes,
α-alkylation with alkyl trifluoroborates.

## Combining catalyst classes

Modern asymmetric catalysis often uses dual-catalyst
systems:

- **Chiral phosphoric acid + chiral amine** (Jacobsen,
  List).
- **Chiral amine + Lewis acid** for Mukaiyama-Michael.
- **Photoredox + organocatalyst** (MacMillan).
- **Photoredox + Ni / Pd + organocatalyst** — three-
  catalyst chemistry frontier.

## Industrial perspective

- **Codexis biocatalysis cascades** — organocatalysis is a
  step within larger biocatalytic routes.
- **Pfizer + Boehringer-Ingelheim** — iminium catalysis in
  process chemistry; benefits from no metal residues.
- **AbbVie + Genentech** — organocatalytic cascades for
  natural-product API synthesis (e.g. eribulin's enamine
  step).
