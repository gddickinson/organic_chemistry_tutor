# Lithium chemistry — directed ortho metalation deeper

The intermediate-tier lesson covered Snieckus's directed
ortho metalation (DoM). This lesson goes deeper into
organolithium chemistry: alkyl-Li reagents, halogen-metal
exchange, fluxional aggregation, and modern flow chemistry.

## The organolithium ladder

```
RLi reagents in order of strength + reactivity:
n-BuLi  <  s-BuLi  <  t-BuLi  <  PhLi  <  MeLi  ≈  vinyl-Li
```

(Strength here = ability to deprotonate; reactivity for
addition follows similar trend.)

- **MeLi**: 1.6 M in Et₂O; most stable organolithium.
- **n-BuLi**: 2.5 M in hexane; standard for DoM.
- **s-BuLi**: 1.4 M in cyclohexane; bulkier; helps regiochem.
- **t-BuLi**: 1.7 M in pentane; **pyrophoric**; fastest +
  riskiest.
- **n-HexLi, n-OctLi**: rare; specialty.
- **Phenyl-Li**: 1.8 M in dibutyl ether; selective aryl
  source.

## Aggregation

Organolithium reagents aggregate in solution. n-BuLi:

```
hexane: hexamer (Bu-Li)₆
THF (with TMEDA): dimer (Bu-Li)₂
```

Smaller aggregates = more reactive. Adding **TMEDA** or
**HMPA** breaks aggregates → enhances kinetic basicity.

## Halogen-metal exchange

```
ArBr + n-BuLi → ArLi + n-BuBr   (fast at -78 °C)
```

A useful way to convert ArBr / ArI into ArLi without
deprotonating other C-H bonds. For ArCl, exchange is slow
+ usually doesn't compete with deprotonation.

Modern variant: **t-BuLi + ArI → ArLi + 2 t-Bu-byproducts**
at -100 °C in flow → very fast + clean (Yoshida's flow
chemistry).

## Lateral lithiation

Alkyl groups α to a benzene ring can be deprotonated by
strong base:

```
Ph-CH₂-CO₂R + LDA → Ph-CH(Li)-CO₂R → enolate-like
                  + RX → α-arylated ester
```

Enantioselective lateral lithiation via Beak chiral
diamine.

## Beak chiral diamines (sparteine, Bauer's diamine)

Add (-)-sparteine (a chiral natural product) to s-BuLi →
chiral organolithium → enantioselective deprotonation /
substitution. Beak (Illinois) showed 80-95 % ee for
α-aminoalkyl substrates.

Sparteine is a natural alkaloid; also synthetic
substitute (Bauer's diamine, Boc-PMP).

## Why Li chemistry is dangerous

- **t-BuLi** is **pyrophoric** in air — must use under inert
  atmosphere. Lab fires + serious injury / death have
  happened. Always handle in glove box or schlenk
  technique.
- **Sealed-flask Li reactions** can build up gas pressure.
- **Quench carefully** — small-scale ROH quench can be
  rapid + exothermic.

## Modern flow Li chemistry

Yoshida (Kyoto) + Jensen + Buchwald scale Li chemistry safely
using microreactors:

- Mix organolithium + electrophile in a flow tube with
  ms-resolved residence time.
- Carry out lithiation at -78 °C in seconds → quench
  immediately → no over-deprotonation.
- Pfizer + Genentech use this for kg-scale ArLi chemistry.

## Common transformations from RLi

| Electrophile | Product |
|--------------|---------|
| RX | R-R' (alkylation) |
| R₂C=O | R-CR'₂-OH (alcohol after workup) |
| R-CO-R' | R-CR'(OH)-R'' (alcohol) |
| R-COOR' | R-COR' (after first attack; LAH-like) |
| R-CONR'₂ | R-CONR'-R'' (after first attack) |
| R-CN | R-CO-R' (after hydrolysis of imine) |
| CO₂ | R-COOH |
| DMF | RCHO |
| B(OMe)₃ | R-Bpin (after pinacol) |
| TMSCl | R-TMS |
| RSSR (disulfide) | R-SR' |
| O₂ | R-OOH (sometimes mess) |
| D₂O | R-D |

## Other organometallic alternatives

- **Organomagnesium (Grignard)**: gentler; tolerates more.
- **Organozinc (Reformatsky, Negishi)**: gentlest; high
  FG tolerance.
- **Organocopper (Gilman)**: soft; 1,4-addition selective.
- **Organocerium (Imamoto)**: like Grignard but less basic;
  useful for hindered substrates.

Order by basicity (descending): RLi > RMgX > R₂Zn ≥ RZnX
> R₂CuLi > Ce reagents.

Order by hardness: RLi > RMgX > R₂Zn > R₂CuLi.
