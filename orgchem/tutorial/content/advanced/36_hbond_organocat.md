# Hydrogen-bonding organocatalysis — Jacobsen thiourea + squaramide

A fundamentally different approach to organocatalysis:
**a chiral H-bond donor** activates the substrate without
forming a covalent intermediate. Eric Jacobsen + Takemoto
+ others pioneered this since ~ 2000.

## The basic idea

Two adjacent N-H bonds on a thiourea (or squaramide) form
a **bidentate H-bond** to a substrate's electrophile (a
carbonyl, an imine, a nitro group):

```
S         R                R
||        \                /
N-H ··· O=C ↔  N-H ··· O=C
|                |
N-H ···          N-H ··· anion or polar group
|                |
chiral pocket → directs prochiral attack of nucleophile
```

The H-bond:

- Activates the carbonyl (or imine, nitro) as a stronger
  electrophile (similar to Lewis-acid activation but
  organocatalytically).
- Sets the substrate orientation in the chiral pocket.

## Famous catalysts

### Jacobsen thiourea (2003)

```
chiral cyclohexane-1,2-diamine derived thiourea +
   peptide-like sidechain → bifunctional catalyst
```

Used for:

- **Asymmetric Strecker** (imine + HCN → α-amino nitrile)
  → > 95 % ee.
- **Acyl-Pictet-Spengler** → > 90 % ee.
- **Asymmetric ring opening** of epoxides + aziridines.
- **Asymmetric Michael** of nitroalkenes.

### Takemoto thiourea

```
1,2-cyclohexanediamine + 3,5-bis(trifluoromethyl)phenyl
   thiourea (one side) + tertiary amine (other side)
```

Bifunctional: thiourea binds the electrophile, tert. amine
deprotonates the nucleophile.

Used for:

- **Asymmetric Henry** (nitroaldol).
- **Asymmetric Michael** of malonates onto nitroalkenes.

### Squaramide (Rawal, 2008)

```
chiral 1,2-amino-cyclohexane diamide of squaric acid →
   tighter NH-NH gap (5.6 Å vs 6.4 Å for thiourea) →
   stronger H-bond binding
```

Squaramides are typically 10× more reactive than
thioureas + give higher ee for many substrates.

## Examples of Jacobsen catalysis in practice

### Acyl-Pictet-Spengler

```
β-arylethylamine + aldehyde + thiourea cat. → tetra-
   hydroisoquinoline (chiral)
```

Used in Cialis (tadalafil) precursor synthesis.

### Strecker hydrocyanation

```
N-Boc imine + TMSCN + Jacobsen thiourea → α-amino nitrile
                                          (> 95 % ee)
```

Industrial route to chiral β-amino acids.

### Polyene cyclisation

```
(Z,Z)-tetraene + thiourea cat. → polycyclic
```

Mimics steroid biosynthesis (cation cascade).

## Brønsted acid catalysis (BINOL phosphates)

Closely related: **chiral Brønsted acid** (BINOL phosphate)
protonates an imine + activates it for asymmetric
attack.

```
N-aryl ketimine + Hantzsch ester (H source) +
                  BINOL phosphate (~ 5 mol %) →
                  chiral amine (> 95 % ee)
```

Akiyama (2004) + Terada (2004) + List + Antilla independently
discovered this.

Used for:

- **Asymmetric reductive amination** (transfer
  hydrogenation).
- **Asymmetric Mukaiyama-Mannich**.
- **Asymmetric pinacol rearrangement**.
- **Asymmetric chlorocyclisation**.

## Modern combinations

Hybrid catalysts combine multiple recognition modes:

- **Thiourea + chiral phosphate** — anion-binding catalysis.
- **Brønsted acid + chiral aminocatalyst** — orthogonal
  activations.
- **Photoredox + chiral Brønsted acid** → hybrid asymmetric
  photocatalysis.

## Industrial scope

H-bond organocatalysis is favoured for:

- **Pharmaceutical chemistry** — no metal residues; cleaner
  filings.
- **Cost** — chiral organic catalysts cost $50-200/g vs
  Rh-BINAP $500+/g.
- **Air + moisture stability** — bench reagents.

But the catalyst loading (5-30 mol %) is higher than metal
asymmetric catalysis (0.1-1 mol %), so scale-up cost
balances out depending on substrate.

## Limits

- Slower than metal catalysis (TON 100-1000 vs 10⁵+).
- Substrate scope: works best on imines, nitro compounds,
  α,β-unsat carbonyls; less general than metal catalysis.
- High loading often needed.
