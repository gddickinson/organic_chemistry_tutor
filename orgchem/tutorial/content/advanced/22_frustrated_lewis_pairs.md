# Frustrated Lewis pairs — metal-free H₂ activation

A **frustrated Lewis pair (FLP)** is a Lewis acid + Lewis
base that **cannot form a classical adduct** (steric clash
prevents it). The unquenched reactivity activates small
molecules — H₂, CO₂, N₂O, alkenes, alkynes — without any
transition metal. Stephan + Erker spurred the field
starting 2006.

## The classical Lewis-pair picture

Normally, a Lewis acid (B(C₆F₅)₃, BCl₃) + a Lewis base
(PR₃, NR₃) form a dative bond:

```
Ph₃P + B(C₆F₅)₃ → Ph₃P-B(C₆F₅)₃   (classical adduct, dead)
```

Once paired, neither retains reactivity. **Steric bulk** —
on either or both partners — prevents adduct formation:

```
Mes₃P + B(C₆F₅)₃ → no adduct (Mes = 2,4,6-trimethylphenyl
                              = mesityl, too bulky)
```

Both Lewis sites remain free + co-located → cooperative
activation of small molecules between them.

## H₂ activation (Stephan, 2006)

```
Mes₃P + B(C₆F₅)₃ + H₂ → [Mes₃P-H]⁺ [HB(C₆F₅)₃]⁻
                          (frustrated → Lewis pair captures H-H)
```

The H-H bond polarises across the FLP gap; one H ends up
on P (as proton) + one on B (as hydride). The product is
an **isolable phosphonium / borohydride salt**.

Heated to 100 °C, the salt releases H₂ + reverts. So FLPs
are a metal-free **catalytic H₂ activation system**.

## FLP-catalysed hydrogenation

After H₂ activation, the Lewis-pair "hydride" + "proton" can
be delivered to a substrate:

```
imine + FLP + H₂ → amine
```

- Substrates: imines, enamines, silyl enol ethers, silyl
  ketene acetals, electron-poor olefins.
- Catalyst loading 5-10 mol % FLP.
- Asymmetric variants — chiral binaphthyl borane + chiral
  phosphine give > 90 % ee for hindered ketones, imines.

This is the first **metal-free catalytic hydrogenation**
that's competitive in scope with Pd / Pt / Rh systems.

## CO₂ activation

Some FLPs reversibly bind CO₂ at room T:

```
FLP + CO₂ → FLP-CO₂ adduct
```

Under H₂ + heat: → formic acid + FLP. This is a route to
metal-free CO₂ → HCOOH conversion (relevant to CO₂ utilisation).

## NO + N₂O activation

NO binds many FLPs to give Lewis-pair-stabilised nitroxide
adducts (FLP-NO·) — first stable persistent NO·-on-Lewis-
acid systems.

N₂O activation gives O-transfer products (mimics flavin
oxidation chemistry).

## Mechanism — an FLP inside a "pocket"

Computation (Pápai, Grimme): in solution, an unbound
phosphine + borane drift around the same solvent cage.
H₂ enters the gap between them; the linear H-H bond
polarises along the P···B vector. Charge separation forms
a transition state with partial P-H bond + partial B-H
bond.

This is **non-classical activation** — neither Lewis site
attacks H₂ alone (P alone wouldn't grab H, B alone neither).
The cooperativity is the chemistry.

## Modern FLP systems

- **Stephan-Erker classical**: bulky phosphine + B(C₆F₅)₃.
- **Intramolecular FLPs** — phosphino-borane (R₂P-link-BR'₂)
  in the same molecule, gives unimolecular kinetics +
  asymmetric scope.
- **Carbene-based FLPs** — NHC + B(C₆F₅)₃ — even more
  reactive.
- **Aluminium-based FLPs** — Al(C₆F₅)₃ + bulky N donor.
- **All-carbon FLPs** (Marder, Power) — emerging, hard
  to control.
- **Enzymatic FLP analogues** (Tilley) — Mn-cluster mimics
  of [Fe-Fe] hydrogenase exhibit FLP-like H₂ heterolysis
  in the active site.

## Limits

- High catalyst loading (~ 10 mol %) compared to Rh / Ir
  hydrogenation.
- FLPs are sensitive to moisture + acid — protic substrates
  protonate the phosphine + kill the system.
- Some substrates (sterically hindered, electron-rich
  alkenes) don't reduce.
- Industrial scale-up still pending (cost of fluorinated
  borane + sensitive setup).

## Why FLPs matter

- **Conceptually unifying** — bridge organometallic +
  classical organic + main-group chemistry.
- **Sustainability** — no precious metals.
- **Mechanistic teaching tool** — shows that
  cooperative-activation is a general principle (also
  exploited by metal-Brønsted-base + metal-NHC systems).
- **Enzyme-mimicry frontier** — many enzymes (hydrogenases,
  formate dehydrogenase) use cooperative metal + base
  activation analogous to FLPs.
