# Controlled radical polymerisation — RAFT, ATRP, NMP

Free-radical polymerisation is the textbook way to make
polystyrene, PMMA, polyacrylates. It's robust + tolerant of
water + functional groups, but **uncontrolled**: the polymer
has broad molecular-weight distribution (Đ ~ 1.5-3), no
defined end-groups, and you can't make block copolymers.

**Controlled / living radical polymerisation (CRP)** fixes
this — narrow MW (Đ ~ 1.05-1.2), defined chain ends, and
on-demand block formation.

## The problem with conventional radicals

Free radicals couple + disproportionate at diffusion-limited
rates (k_t ~ 10⁸ M⁻¹s⁻¹). At equilibrium [P•] ~ 10⁻⁷ M, with
[P-H] ~ 1 M; chains live ~ 1 sec before termination. You
cannot direct chain end-fidelity.

## The CRP solution: equilibrium

Add a **dormant species** in equilibrium with the active
radical:

```
P-X  ⇌  P•  +  X
   (dormant)   (active)
```

Keep [P•] ~ 10⁻⁹ M (1000 × lower than uncontrolled radical
polymerisation) → termination rate (which scales as [P•]²)
drops 10⁶ × → chains stay alive long enough for sequential
monomer addition.

Three big CRP families implement this differently.

## NMP — nitroxide-mediated polymerisation

Hawker, Georges (1990s). The dormant species is an alkoxyamine:

```
P-O-NR₂  ⇌  P•  +  •O-NR₂   (TEMPO, SG1)
```

- **TEMPO** — first generation; works for styrene only at
  130 °C.
- **SG1, TIPNO** — second generation; works for styrene +
  acrylates at 100-120 °C.
- **No metals** — clean for biomedical applications.
- Limited: doesn't work well for methacrylates (TEMPO-Me too
  fast).

## ATRP — atom-transfer radical polymerisation

Matyjaszewski + Sawamoto (1995). The dormant species is an
alkyl halide:

```
P-X + Cu(I)L  ⇌  P• + Cu(II)X-L
```

- **Catalyst**: Cu / pyridine or amine ligand (Me₆TREN, PMDETA,
  TPMA).
- **Initiator**: alkyl halide (ethyl 2-bromoisobutyrate,
  benzyl chloride).
- **Monomer scope**: very wide — styrenes, acrylates,
  methacrylates, acrylonitrile, vinylpyridines.
- **Modern variants** to reduce Cu loading:
  - **ARGET-ATRP** (activator regenerated by electron
    transfer) — uses ascorbic acid to keep Cu(I) topped up;
    ppm-level Cu.
  - **eATRP / photoATRP** — electrochemical / photoredox
    activator regeneration.
  - **SI-ATRP** — surface-initiated; densely tethered
    polymer brushes.
- **Drawback**: Cu residues a problem for medical /
  biological applications.

## RAFT — reversible addition-fragmentation chain transfer

Rizzardo, Moad (CSIRO, 1998). The dormant species is a
thiocarbonylthio compound (dithioester, trithiocarbonate,
dithiocarbamate, xanthate):

```
P• + S=C(R)-S-R'  ⇌  P-S-C(R)(•)-S-R'  ⇌  P-S-C(R)=S + •R'
                   (intermediate radical)
```

The **R group** acts as the "leaving radical" + initiates
the next chain; the **Z group** stabilises the intermediate.

- **Z group selection** based on monomer:
  - Phenyl (dithiobenzoate) — methacrylates, styrene.
  - SR'' (trithiocarbonate) — acrylates, acrylamides,
    styrene.
  - OR'' (xanthate) — vinyl acetate, less-activated
    monomers.
- **Excellent functional-group tolerance** — water,
  protic solvents, near room temperature.
- **No metals** — biomedical-friendly.
- **Drawback**: chain ends carry sulfur (smelly for
  high-Đ samples; aminolysis converts to thiol → useful
  bioconjugation handle).

## Block copolymers

CRP makes block-copolymer synthesis routine:

1. Polymerise monomer A → A-P-X with all chains carrying
   the dormant end-group.
2. Add monomer B → continue polymerisation → A-block-B-P-X.
3. Repeat for triblock, multiblock, gradient.

Industrial relevance:

- **Pluronic surfactants** — PEO-PPO-PEO triblocks (BASF).
- **Kraton thermoplastic elastomers** — polystyrene-poly-
  butadiene-polystyrene SBS (anionic-living, but same
  conceptual approach).
- **Self-assembling nanoparticles** — amphiphilic AB blocks
  → micelles + vesicles for drug delivery.

## Other CRP families

- **OMRP** (organometallic-mediated) — Co, Ti, Mo
  alkyls.
- **RDRP via tellurides + bismuthides** (Yamago).
- **Photoinduced CRP** — Hawker + Boyer photoATRP /
  PET-RAFT — visible-light spatial + temporal control.
- **Iron-catalysed ATRP** (Magenau, Schubert) — earth-
  abundant alternative.

## Living anionic polymerisation

The original "living" polymerisation (Szwarc, 1956). Carbanion
ends never terminate (no free anion). Limitations: no protic
solvents + no protic functional groups (hydroxyls, amines,
acids). Used commercially for SBR rubber, butyl rubber,
controlled-architecture diblocks (Pluronic, Kraton).
