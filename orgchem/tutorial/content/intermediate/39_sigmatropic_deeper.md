# Sigmatropic rearrangements deeper — Claisen variants

The pericyclic-intro lesson covered [3,3]-sigmatropic
basics. This lesson maps the modern Claisen-rearrangement
toolbox + its synthetic uses.

## [3,3]-sigmatropic — the framework

Six-electron concerted pericyclic; chair-like or boat-like
6-membered TS; thermally allowed (4n+2, n=1).

```
1  2  3
 \ | /
  X        →    1'  2'  3'
  |               \ | /
  Y                Y'
  |
6  5  4
```

A σ-bond between atoms 1 + 1' migrates to a new σ-bond
between atoms 3 + 3', with shift of the connecting π
bonds.

## Cope rearrangement

```
1,5-hexadiene  ⇌  1,5-hexadiene (atoms swap)
```

C-C σ-bond migrates over a 6-atom all-C system.
Symmetric; equilibrium = 50/50 unless biased by
substitution.

**Anionic Cope** (Evans 1975) — α-OH on C3 → after
deprotonation, the rearrangement is much faster
(Evans-Mislow). Used to forge complex C-C frameworks.

**Oxy-Cope** — same with oxide on C3; rate enhancement
~ 10¹⁰ × the neutral version.

## Claisen rearrangement (thermal, 1912)

```
allyl vinyl ether  →  γ,δ-unsaturated carbonyl
CH₂=CH-CH₂-O-CR=CHR'  →  CH₂=CH-CH₂-CR(R'CHO) (or similar)
```

Aliphatic Claisen: simple allyl vinyl ether. The chair-
like TS controls stereochemistry — defined Z- or E-enol-
ether → defined diastereomer of product.

## Claisen variants (the synthesis kit)

### Aromatic Claisen

```
aryl allyl ether (ArO-CH₂-CH=CH₂) + heat (~ 200 °C) →
                ortho-allyl phenol (after enol tautomerisation)
```

If both ortho positions blocked, the allyl group jumps to
the para position (via two consecutive [3,3] then [1,3]
shifts). Lewis-acid catalysis lowers T (Mikami used
chiral Lewis acid for asymmetric aryl-Claisen).

### Ireland-Claisen (1972)

```
allyl ester + LDA + TMSCl → silyl ketene acetal +
        Δ → carboxylic acid (after silyl removal)
```

Strong + selective. Z- or E-silyl ketene acetal is set by
LDA/HMPA conditions → controls product stereochemistry.

Used heavily in Corey's prostaglandin synthesis +
modern total synthesis of Polyketides.

### Johnson-Claisen (1970)

```
allyl alcohol + orthoester (CH₃C(OEt)₃) + acid catalyst →
                γ,δ-unsaturated ester
```

Avoids the need to pre-form a vinyl ether. Easy to scale.

### Eschenmoser-Claisen (1964)

```
allyl alcohol + N,N-dimethylacetamide dimethyl acetal
                + heat → γ,δ-unsat amide
```

Like Johnson but gives an amide. Convenient when the
amide is the target.

### Carroll rearrangement

```
allyl β-ketoester → γ,δ-unsat ketone + CO₂
                  (decarboxylative Claisen variant)
```

Pd-catalysed asymmetric Carroll (Stoltz, Tunge) →
quaternary stereocentres in one step.

### Bellus-Claisen (allyl-lithio)

```
allyl ester + LDA → enolate → [3,3] → α-allyl ester
```

For allyl esters of simple carboxylic acids; gives α-
allylated products.

## Aza-Claisen + thio-Claisen

Replace the central O with N (aza-Claisen) → γ,δ-unsat
amine; or with S (thio-Claisen) → γ,δ-unsat thio-
ether. Different rates, different selectivities.

Aza-Claisen of N-allyl enamines = key step in some
alkaloid syntheses.

## Stereochemistry — chair-like vs boat-like

Most Claisen variants go through a chair-like 6-membered
TS:

- Substituents prefer **equatorial** positions to minimise
  diaxial strain.
- Predicts cis vs trans of new C-C / C-O bonds.

For locked systems (e.g. cyclohexenyl), the boat may be
forced, giving the opposite stereochemistry.

## Sigmatropic + organocatalysis

Recent: chiral H-bond donors (Jacobsen thiourea, BINOL
phosphate) + ionic intermediates (oxocarbeniums, iminium)
catalyse asymmetric Claisen variants → > 90 % ee.
