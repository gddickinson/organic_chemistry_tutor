# Beckmann + Schmidt + Curtius — three migrations to nitrogen

These three named rearrangements share a common motif:
**a 1,2-shift to an electron-deficient nitrogen** (nitrene
or nitrenoid). Each is a workhorse for installing C-N
bonds.

## Beckmann rearrangement

```
R₂C=N-OH + acid (PCl₅, SOCl₂, or conc. H₂SO₄, or P₂O₅)
       → R-CO-NHR' (amide)
```

- Substrate: an oxime (from R₂C=O + H₂N-OH).
- The group **anti to OH** migrates to N.
- Net: ketone → amide.
- Industrial: cyclohexanone oxime → caprolactam (Nylon-6
  monomer; > 4 million tonnes / yr).

### Mechanism

1. Activate OH as a leaving group (-OH₂⁺ or -OTs).
2. Anti-group migrates to N as -OH₂ leaves.
3. Forms nitrilium ion R-C≡N⁺-R' (sp linear).
4. Water attacks C → tetrahedral.
5. Tautomerisation → amide.

### Asymmetric Beckmann

Some chiral H⁺-Brønsted-acid catalysts give
enantioselective Beckmann on prochiral oximes (Yamamoto +
Akiyama).

### Industrial twist

Modern caprolactam plants use vapour-phase Beckmann at 350
°C with a TS-1 (zeolite + Ti) catalyst — completely
liquid-acid-free, dramatically reduced waste vs the
classical sulfuric-acid Beckmann.

## Schmidt reaction

```
R₂C=O + HN₃ + H⁺ → R-CO-NHR' + N₂
```

- Substrate: ketone (or carboxylic acid → tetrazole, in a
  variant).
- HN₃ is the source of N (and N₂ is lost as a leaving
  group).
- Migration anti-period to leaving N₂.

### Mechanism

1. Protonate ketone → oxocarbenium.
2. HN₃ attacks → tetrahedral.
3. Loss of H₂O → R₂C=N⁺-N₂ → loses N₂ as the migration
   happens.
4. Nitrilium → water → amide.

Larger group migrates preferentially (more substituted →
better σ donor).

### Variant: Aubé asymmetric Schmidt

Aubé (KU, 1990s) — intramolecular Schmidt of azide
tethered to a ketone → ring-expansion + N installation in
one step. Used in alkaloid synthesis.

## Curtius rearrangement

```
R-CO-OH → R-CO-Cl → R-CO-N₃ → (Δ) → R-N=C=O + N₂
                          (acyl azide)  (isocyanate)
+ ROH → R-NH-CO-OR (carbamate)
+ H₂O → R-NH₂ + CO₂↑ (1° amine)
```

- Substrate: carboxylic acid (via acyl azide).
- N source is the azide nitrogen.
- Net: COOH → NH₂ with loss of one C (as CO₂).
- Often performed in one pot with **DPPA** (diphenyl
  phosphoryl azide) as direct conversion of acid to acyl
  azide.

### Why Curtius is favoured over Schmidt + Beckmann

- **Mild conditions** — heat alone (or boiling t-BuOH).
- **One-pot** — DPPA + Et₃N + alcohol → Boc-protected amine
  in one step.
- **Stereochemistry retained** at the migrating carbon
  (concerted shift).

### Bridges to total synthesis

The acyl azide → isocyanate sequence enables installing a
carbamate-protected amine directly on a complex framework
without manipulating the molecule's other functional
groups.

## Hofmann rearrangement

```
R-CO-NH₂ + Br₂ + NaOH → R-NH₂ + CO₂↑
```

Same family — nitrogen-bound bromide is the leaving group,
the alkyl group migrates from C to N. Net: amide → 1°
amine with loss of one C.

### Mechanism

1. NaOH deprotonates NH₂; Br⁺ adds → R-CO-NHBr.
2. Loss of Br⁻ from N + R migration → isocyanate.
3. Hydrolyse → carbamate (transient) → CO₂ + amine.

## Lossen rearrangement

```
R-CO-NH-OH (hydroxamic acid) + acid (or activator) →
                                R-N=C=O (isocyanate)
```

Hydroxamic acid → isocyanate via N-acyloxy intermediate.
Like Curtius but starts from a hydroxamic acid.

## Choosing among them

| Starting material | Use |
|-------------------|-----|
| Ketone → amide | Beckmann (oxime route) or Schmidt (HN₃) |
| Carboxylic acid → 1° amine | Curtius (DPPA) — most popular |
| Carboxylic acid → carbamate | Curtius + ROH trap |
| Amide → 1° amine | Hofmann |
| Hydroxamic acid → isocyanate | Lossen |
