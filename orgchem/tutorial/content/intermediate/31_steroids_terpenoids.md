# Steroids + terpenoids — biosynthesis + structure

A huge fraction of natural products + pharmaceuticals share
a common biosynthetic origin: the **isoprene** unit
(C₅H₈). Counting C in multiples of 5 gives the family
classification + tells you the biosynthetic pathway.

## The isoprene rule (Ruzicka)

Natural-product carbon skeletons are built from C₅
isoprene-equivalent units — almost universally:

| Class | C count | Examples |
|-------|---------|----------|
| **Hemiterpene** | C₅ | isoprene, prenol, dimethylallyl |
| **Monoterpene** | C₁₀ | menthol, limonene, geraniol, α-pinene, citronellal |
| **Sesquiterpene** | C₁₅ | farnesol, artemisinin, β-caryophyllene |
| **Diterpene** | C₂₀ | retinol (vit A), gibberellic acid, taxol skeleton |
| **Sesterterpene** | C₂₅ | rare; manoalides, ophiobolins |
| **Triterpene** | C₃₀ | squalene, cholesterol-precursor; betulinic acid; lanosterol |
| **Tetraterpene** | C₄₀ | β-carotene, lycopene |
| **Polyterpene** | (C₅)ₙ | natural rubber (cis-poly-isoprene) |

## The biosynthesis

### Two routes to isopentenyl pyrophosphate (IPP)

- **Mevalonate pathway** (animals, fungi, archaea, cytosol
  of plants):
  ```
  3 acetyl-CoA → HMG-CoA → mevalonate → IPP + DMAPP
  ```
  (HMG-CoA reductase is the rate-limiting step + the
  statin drug target.)
- **MEP pathway** (most bacteria, plant chloroplasts):
  ```
  pyruvate + glyceraldehyde-3-phosphate → DXP → MEP → IPP + DMAPP
  ```

### IPP + DMAPP polymerise

```
IPP + DMAPP → GPP (geranyl-PP, C10) → FPP (farnesyl-PP,
              C15) → GGPP (geranylgeranyl-PP, C20)
                  → squalene (C30, head-to-head FPP+FPP)
```

Each elongation by *prenyl transferase* (FPP synthase, etc.)
adds one C₅ unit. Stereochemistry: trans-double bonds for
farnesyl; cis for natural rubber.

### Cyclisation

Linear isoprenyl pyrophosphates fold + cyclise into rings
via terpene cyclases — carbocation cascade chemistry:

```
GPP → bornyl cation, pinene, limonene cation, terpineol cation, ...
FPP → caryophyllene, germacrene A, eudesmane skeletons
GGPP → tax-11-en-cation → taxol skeleton; gibberellins
Squalene → (oxidosqualene) → lanosterol → cholesterol → bile / hormones
```

The **oxidosqualene → lanosterol** step performs ~ 8
sequential 1,2-shifts in one enzyme active site. Eschenmoser
+ Stork independently proposed the mechanism in 1955; X-ray
of cyclase confirmed in 2007.

## Steroid skeleton

The **gonane** core: 4 fused rings (3 × six + 1 × five),
17 carbons. Substituents:

- **Cholesterol** — animal, C-27. Membrane component +
  precursor.
- **Ergosterol** — fungal sterol, target of azole
  antifungals.
- **Sitosterol / stigmasterol** — phytosterols.
- **Bile acids** (cholic, deoxycholic, chenodeoxycholic) —
  detergents for fat digestion.
- **Steroid hormones**:
  - **Pregnane** (C-21) — progesterone, cortisol,
    aldosterone.
  - **Androstane** (C-19) — testosterone, DHT.
  - **Estrane** (C-18) — estradiol (loses C-19, aromatic A
    ring).
- **Vitamin D₃** — secosteroid (B-ring opened) from 7-
  dehydrocholesterol + UV.
- **Cardiac glycosides** — digoxin, digitoxin (Na⁺/K⁺-ATPase
  inhibitors).

## Famous monoterpenes

- **Menthol** — peppermint; cooling sensation via TRPM8 ion
  channel agonism.
- **Limonene** — (R)-form smells like orange, (S)-form like
  pine — chirality in nature.
- **Camphor** — Bornyl/camphor cation cyclisation.
- **α-Pinene** — pine resin; major component of turpentine.
- **Geraniol** — rose; widely used in perfumery.
- **Citral** — lemon; food + fragrance flavouring.

## Famous sesquiterpenes

- **Artemisinin** — endoperoxide from Artemisia annua;
  antimalarial, 2015 Nobel (Tu Youyou). Worldwide standard
  for falciparum malaria.
- **β-Caryophyllene** — clove oil; CB2 receptor agonist.
- **Farnesol** — lily-of-the-valley odor.
- **Trichothecenes** — Fusarium toxins; protein-synthesis
  inhibitors.

## Famous diterpenes

- **Retinol (Vitamin A)** + **retinal** + **retinoic acid**
  — vision + epithelial differentiation.
- **Gibberellic acid (GA₃)** — plant hormone; agriculture
  application as growth promoter.
- **Taxol (paclitaxel)** — Pacific yew; tubulin-stabilising
  anti-cancer drug.
- **Forskolin** — coleus; activates adenylate cyclase.
- **Phorbol esters** — protein-kinase-C activators; tumour-
  promoter probes.

## Tetraterpenes

- **β-Carotene** — orange / red; cleaved → 2 × retinol
  (vitamin A precursor).
- **Lycopene** — tomatoes; antioxidant.
- **Astaxanthin** — krill / salmon flesh; potent
  antioxidant.

## Modern asymmetric synthesis

Total synthesis of complex terpenes drives method
development:

- **Taxol** — Nicolaou + Holton groups (1994) — 30+ steps
  each. Industrial route: semisynthesis from baccatin III
  isolated from Pacific yew needles.
- **Vinblastine, Vincristine** — vinca alkaloids;
  Boger + others did total synthesis; original commercial
  route stays semi-synthesis.
- **Artemisinin** — Sanofi's photochemical-flow route from
  artemisinic acid (yeast-fermented); kg-scale flow
  photochemistry.
- **Maoecrystal V, Ingenol, Vinigrol** — recent total
  synthesis benchmarks.
