# The Ene reaction

The **ene reaction** (Hans Alder, 1943) is a concerted
addition of an alkene + an "enophile" (alkene, alkyne,
carbonyl, or other π system) with **migration of an
allylic H**.

## The textbook ene

```
H            H
 \  /         \  /
  C=C   +     C=C    →    new σ-C-C + new C-H  +  shifted C=C
 / \           ↑\
   `H'         /
    \        /
     \      /
      C=C-X (enophile)
```

The H on the allylic position transfers to the enophile;
a new σ-C-C bond forms between the alkene + enophile. The
double bond shifts.

Six-electron concerted; 5-membered TS; allowed thermally.

## Two flavours

### Carbonyl-ene (oxa-ene)

```
RCH=CHR' (alkene) + R''₂C=O (enophile, carbonyl) →
                  → R'CH(OH)CR''₂-CHR (homoallyl alcohol)
```

The C=O is the enophile; the alkene's α-H ends up on O →
homoallyl alcohol.

Lewis-acid-catalysed (SnCl₄, AlCl₃, EtAlCl₂, BF₃·OEt₂);
chiral Lewis-acid catalysts (Ti-BINOL, BINAP-Pd) → highly
enantioselective version.

### Carbon-ene

```
alkene (no C=O) + alkene (electron-poor, enophile) →
                  → larger alkene
```

Slower; needs strong activation. Dehydrogenative ene
variant connects two unactivated alkenes via Pd / Pt /
photoredox.

## Famous applications

### Conia-ene

```
β-ketoester + alkene/alkyne in same molecule + heat or
   metal catalyst → cyclisation forming a new ring
```

The β-ketoester's enol form acts as the alkene; the
distant alkene/alkyne is the enophile. Toste (2004) +
others use Au cat for catalytic asymmetric Conia-ene
cyclisation.

### Schenck ene (with singlet O₂)

```
alkene + ¹O₂ → allyl hydroperoxide (ROOH) + shifted C=C
```

Photochemical generation of ¹O₂ (rose bengal + O₂ +
visible light); ene with the alkene → allyl
hydroperoxide; reduction → allyl alcohol.

Used in artemisinin synthesis (the artemisinin endoperoxide
comes from this in Sanofi's flow process).

### Alder-ene (heat-driven, no metal)

Original Alder discovery; works on activated enophiles
(maleic anhydride, methyl acrylate). Limited substrate
scope at "thermal-only" condition.

## Asymmetric metal-catalysed ene

Mikami's catalytic asymmetric ene (Ti-BINOL Lewis acid +
glyoxylate enophile; 1990s) — gave > 90 % ee for
homoallyl alcohols. Set the stage for asymmetric variants.

Modern: Ru(II), Au(I), Pt(II), Pd(II) all catalyse
intramolecular + intermolecular ene with chiral ligand.

## Comparison with Diels-Alder

| | Diels-Alder | Ene |
|----|-------------|-----|
| Components | 4π diene + 2π dienophile | 2π + allylic-H + 2π |
| TS electrons | 6 (suprafacial, all sigma + π) | 6 (sigma H migration + π changes) |
| Conditions | thermally allowed | thermally allowed |
| New bonds | 1 σ (in ring closure) + 0 σ → 2 σ | 1 σ + 1 σ-H + π shifts |
| Catalyst | no catalyst usually; Lewis acids accelerate | usually requires Lewis acid / metal |
| Stereochemistry | strict endo / exo | strict (TS chair-like) |

## Limits

- **Slow without catalyst** for unactivated substrates.
- **Diels-Alder competes** when diene + dienophile are
  present.
- **Selectivity** can be poor on regiochemistry grounds for
  unsymmetric substrates.
