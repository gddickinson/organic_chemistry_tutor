# Mannich + heteroatom-Mannich variants

The **Mannich reaction** combines a carbonyl, an amine,
and an enolisable α-C-H component into a β-amino carbonyl.
Together with aldol it is the canonical C-C bond-formation
strategy.

## Classical Mannich (1912)

```
formaldehyde + secondary amine + ketone (with α-H)
→ β-amino ketone (Mannich base)
```

Mechanism:

1. Amine + HCHO → iminium ion (CH₂=NR₂⁺) (alone or in
   solution).
2. Ketone + acid → enol or enolate.
3. Enol attacks iminium → β-amino ketone.

The product is a versatile intermediate — easily converts
to α,β-unsaturated ketones (Mannich-then-elimination),
amino-alcohols (reduction), or N-methylated amines
(Eschweiler-Clarke variant).

## Heteroatom-Mannich variants

The "iminium + nucleophile" template generalises to many
nucleophile classes.

### Aza-Mannich (azomethine + Si-enolate)

```
N-protected α,β-imine + silyl enol ether + Lewis acid
→ β-amino carbonyl + retro-Si
```

Acceptable nucleophiles: silyl enol ethers, silyl ketene
acetals, allyl-Sn, vinyl-Si, allenyl-Sn — any soft
nucleophile.

### Pictet-Spengler

```
β-arylethylamine + aldehyde + acid →
   tetrahydroisoquinoline or β-carboline
```

A Mannich-Friedel-Crafts hybrid. Iminium ion forms with
the aldehyde; an electron-rich arene on the amine
substrate attacks intramolecularly → 6-membered ring.

Used to synthesise:

- **Tetrahydroisoquinoline** alkaloids (apomorphine,
  emetine).
- **β-Carboline** alkaloids (yohimbine, reserpine).
- **Tetrahydro-β-carbolines** (tadalafil — Cialis).

Asymmetric variants: List + Jacobsen chiral acid catalysts
→ > 95 % ee.

### Pictet-Spengler in biology

Several plant alkaloids (papaverine, berberine, morphine)
incorporate Pictet-Spengler-derived cores from
biosynthetic L-DOPA + dopamine.

### Bischler-Napieralski

Closely related to Pictet-Spengler; uses POCl₃ or P₂O₅
to drive cyclodehydration of β-arylethylamides → 3,4-
dihydroisoquinolines.

### Strecker amino-acid synthesis

```
RCHO + NH₃ + HCN → R-CH(NH₂)-CN → R-CH(NH₂)-COOH
                  (α-aminonitrile)  (after hydrolysis)
```

A Mannich-like (aldehyde + amine → iminium) but with HCN
as the nucleophile. Generates racemic amino acids;
asymmetric versions use chiral imines (Ellman's
sulfinamide).

### List asymmetric Mannich

```
α,β-unsaturated aldehyde + amino-aldehyde + chiral amine cat.
                          → β-amino ketone with > 90 % ee
```

Proline + MacMillan's imidazolidinone catalysts.

## Modern multi-component reactions (MCRs)

The Mannich extends to 3+ component reactions in one pot:

### Ugi 4CR

```
amine + aldehyde + carboxylic acid + isocyanide
→ α-acylaminoamide (one step, all four components)
```

Generates large libraries quickly. Used in pharma SAR +
DEL synthesis.

### Passerini 3CR

```
carboxylic acid + aldehyde + isocyanide → α-acyloxyamide
```

Older (1921); simpler 3CR.

### Hantzsch dihydropyridine synthesis

```
2 β-ketoester + aldehyde + NH₃ → dihydropyridine
```

Scaffold for nifedipine, amlodipine + many calcium channel
blocker drugs.

## Industrial relevance

- **Diphenhydramine (Benadryl)** — α-amino-aryl ether via
  Mannich then etherification.
- **Tropacocaine + tropine alkaloids** — Robinson's 1917
  one-pot Mannich + dehydration.
- **Tadalafil (Cialis)** — Pictet-Spengler key step.
- **Atorvastatin (Lipitor)** — Stetter-aldol + asymmetric
  steps; Mannich-like intermediates.
