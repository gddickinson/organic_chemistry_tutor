# Wolff rearrangement + Arndt-Eistert homologation

The **Wolff rearrangement** converts an α-diazo ketone into
a ketene, with migration of one substituent to the carbene
C. **Arndt-Eistert** is the use of Wolff for chain
homologation — adding one carbon between the carbonyl + R.

## Wolff rearrangement (1902)

```
R-CO-CHN₂  →  (loss of N₂)  R-C(=carbene)-CHO  →  rearrange  →  R-CH=C=O (ketene)
```

The α-carbon (bearing the diazo) becomes a singlet carbene
on N₂ loss. R migrates from the carbonyl C to the carbene C
→ ketene.

Initiation: heat, UV light, or transition-metal catalyst
(Rh, Ag, Cu).

### Migratory aptitude

H > aryl > alkyl. Like Beckmann + Schmidt, the migrating
group's stereochemistry at C is retained.

## Arndt-Eistert homologation (1935)

The classic 3-step sequence to convert a carboxylic acid
into a one-carbon-extended carboxylic acid:

```
Step 1: R-COOH + (COCl)₂ → R-COCl
Step 2: R-COCl + 2 CH₂N₂ (excess) → R-CO-CHN₂ + N₂↑ + CH₃Cl
Step 3a: R-CO-CHN₂ + Ag₂O / H₂O → R-CH₂-COOH (one-C homolog)
Step 3b: + ROH → R-CH₂-COOR' (ester)
Step 3c: + RNH₂ → R-CH₂-CONHR' (amide)
```

Net: COOH → CH₂COOH (added one C between R and COOH).

### Why it matters

For a target with a specific carbon skeleton, sometimes
the most direct retrosynthesis adds carbons via Arndt-
Eistert. Alternative one-C extensions: cyanide / hydrolyse,
malonic ester synthesis (adds 2 C with -COOH lost),
Michael addition.

### Modern variants

- **TMSCHN₂** instead of CH₂N₂ — much safer (CH₂N₂ is
  shock-explosive; TMSCHN₂ is a stable yellow oil).
  Workhorse for Arndt-Eistert in the 21st century.
- **Flow chemistry** — generate CH₂N₂ in situ + use it
  immediately; never accumulate dangerous quantities.
- **Photo-Wolff** — UV light at λ < 350 nm initiates Wolff
  on stable α-diazo ketones; tolerant of base + many FGs.

## Asymmetric Wolff

Doyle, Davies + others use chiral Rh(II) carboxylate
catalysts for asymmetric carbene generation from α-diazo
carbonyls. Reactions:

- **Cyclopropanation** of alkenes.
- **C–H insertion** at specific sites.
- **Wolff rearrangement** controlled to give specific
  enantiomers of the resulting acid.

## Bamford-Stevens — generate diazo from tosylhydrazone

```
R-CO-R' → R-CH(=N-NHTs) (tosylhydrazone)
       + base → R-CH(=N⁻N=Ts → Ts⁻ + R-CH=N₂ → carbene/alkene
```

Avoids handling diazomethane directly; gives access to
alkenes (loss of N₂ + 1,2-H shift) or carbenes (which
can do C–H insertion or cyclopropanation).

## Hazards

- **Diazomethane (CH₂N₂)** — yellow gas; bp -23 °C; SHOCK +
  HEAT sensitive; Teflon glassware only (no ground glass!).
  Only use freshly prepared in small quantities.
- **TMSCHN₂** — stable in hexane / Et₂O; commercial; SAFE
  for routine lab use. Modern default.
- **Acyl azides** — heat-sensitive, can detonate; > 50 % N
  by mass is over the limit; use solution only.
