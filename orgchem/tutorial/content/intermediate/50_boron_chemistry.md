# Boron chemistry intro

Boron (Group 13) is electron-deficient: its valence shell
has only 6 electrons (sp² hybrid + empty p orbital). This
makes BR₃ compounds Lewis acids, pivotal in many
synthesis tools.

## Borane (BH₃)

```
2 BH₃ ⇌ B₂H₆ (diborane, dimer)
```

Free BH₃ is rare; it dimerises to diborane in pure form.
Stable as borane-THF (BH₃·THF), borane-DMS (BH₃·SMe₂),
borane-pyridine, borane-amine adducts.

### Hydroboration

```
BH₃ + 3 RCH=CHR' → (R'CH(BH(CHR-CH-R')₂)) → trialkylborane
```

Three alkene additions per BH₃; gives trialkylborane.
**Anti-Markovnikov + syn** addition.

For mono-addition use bulky boranes (9-BBN, disiamylborane,
thexylborane) → only one alkene adds.

### Hydroboration-oxidation

```
RCH=CH₂ + BH₃ then H₂O₂/NaOH → RCH₂CH₂OH (anti-Markov.)
```

Workhorse for converting terminal alkenes into primary
alcohols with retention of configuration. Already covered
in lesson 28; repeated for context.

## Boronate esters

```
R-B(OH)₂ → R-Bpin (pinacol boronate, R-B(OCMe₂)₂)
         → R-Bcat (catechol boronate, R-B(OC₆H₄O))
         → R-BMIDA (Burke MIDA boronate, slow-release)
```

Boronate esters are the air-stable form for storage +
handling. Burke MIDA boronates are slow-hydrolysis — used
for **iterative cross-coupling** (Burke 2014).

### Suzuki coupling (recap)

```
R-Bpin + Ar-X + Pd cat. + base → R-Ar
```

The 2010 Nobel-prize chemistry. Tolerates many functional
groups; widely used in pharma + material synthesis.

## Boranes in reduction

- **NaBH₄** — sodium borohydride; mild reducer; aldehyde +
  ketone → alcohol.
- **LiBH₄** — Li counter-ion makes it stronger; reduces
  esters too.
- **NaBH(OAc)₃** — selectively reduces iminium over carbonyl;
  reductive-amination workhorse.
- **9-BBN** — bulky, selective hydroborations (less
  hindered alkene only).
- **Selectride™** (Li tri-sec-butylborohydride) — bulky,
  stereoselective ketone reduction.
- **K-Selectride** + **L-Selectride** — different counter-ions
  give different selectivities.

## Boron in asymmetric synthesis

### CBS reduction

```
prochiral ketone + BH₃ + chiral oxazaborolidine catalyst
→ chiral alcohol (> 95 % ee)
```

Corey-Bakshi-Shibata (1987). Pre-form the catalyst
(commercial); add BH₃·SMe₂ + ketone → enantioselective
reduction.

### Brown asymmetric hydroboration

```
RCH=CHR' + (Ipc)₂BH (diisopinocampheyl borane) →
   chiral alkylborane → oxidation → chiral alcohol (> 95 % ee)
```

Uses (R)- or (S)-α-pinene as the chiral source; cheap
chirality.

### Allylboration (Brown, 1980s)

```
allyl-(Ipc)₂B + RCHO → chiral homoallyl alcohol
```

> 95 % ee for many substrates. Crimmins / Roush / Brown
asymmetric allylation.

### Matteson homologation

```
α-haloboronate + nucleophile → boronate with one extra C
```

Powerful chiral C-extension chemistry. Each cycle adds one
C with controlled stereochemistry.

## Boronic acid + diol equilibria

```
R-B(OH)₂ + 2 HO-R' ⇌ R-B(OR')₂ + 2 H₂O    (boronate ester)
```

**Reversible** + pH-dependent. Modern uses:

- **Glucose sensors** (continuous glucose monitors) — boronic
  acid binds the diol of glucose at pH 7.4.
- **Cancer drug bortezomib** — boronic-acid proteasome
  inhibitor; reversibly binds the catalytic threonine OH
  on the proteasome's β5 subunit.

## Boron-containing drugs

- **Bortezomib (Velcade)** — Multiple myeloma; reversible
  proteasome inhibitor.
- **Ixazomib (Ninlaro)** — oral analogue.
- **Tavaborole (Kerydin)** — antifungal toenail treatment;
  benzoxaborole leucyl-tRNA-synthetase inhibitor.
- **Crisaborole (Eucrisa)** — atopic dermatitis; PDE4
  inhibitor.

Boron is "the next halogen" — many medicinal chemists are
exploring borate isosteres + benzoxaborole scaffolds.
