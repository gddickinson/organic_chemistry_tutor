# Reformatsky reaction + zinc-enolate chemistry

The **Reformatsky reaction** is the zinc-mediated aldol-
type reaction between an α-haloester + a carbonyl.

## The classic reaction

```
α-bromoester (BrCH₂CO₂R) + carbonyl (R'₂C=O) + Zn dust
→ β-hydroxy ester (R'₂C(OH)-CH₂CO₂R)
```

Mechanism:

1. **Zn insertion** into C-Br → α-zinc ester (organozinc
   carboxylate; "Reformatsky reagent").
2. Carbonyl attack — Zn-C is a soft nucleophile, attacks
   the C=O carbon.
3. Workup → β-hydroxy ester.

## Why it's useful

- **Mild** — Zn is much less reactive than Mg or Li → no
  reduction of C=O to alcohol (which Grignards / RLi can
  do).
- **Tolerates esters, amides, alkyl halides** — Zn doesn't
  attack them.
- **Cheap + scalable** — Zn dust + α-bromoester are
  bench reagents.
- **Functions as a soft enolate** — selective for direct
  C-C bond formation.

## Comparison to Aldol + Mukaiyama

| Method | Reagent | Conditions | Selectivity |
|--------|---------|------------|-------------|
| Aldol (NaOH) | enolate from base + carbonyl | aq. base, rt | E-selective often |
| Aldol (LDA) | LDA + carbonyl + RCHO | THF, -78 °C | kinetic enolate; high yield |
| Mukaiyama | silyl enol ether + Lewis acid + RCHO | DCM, -78 °C | Z-enol ether → syn aldol |
| Reformatsky | α-haloester + Zn + RCHO | THF or benzene, rt | mild; limited stereoselection |

## Conditions

- **Solvent**: THF, benzene, ether (anhydrous; Zn-organics
  air-sensitive).
- **Activator**: Zn dust (commercial; activate with HCl
  wash + dry); or use Zn(Cu) for difficult substrates.
- **Temperature**: rt to reflux.
- **Substrates**: α-Br ester + aldehyde or ketone; methyl
  + ethyl + benzyl esters all work.

## Modern Reformatsky variants

### Honda-Reformatsky

Use **Et₂Zn / Reformatsky reagent** for cleaner reactions
+ asymmetric variants. Et₂Zn forms a more-defined zinc
enolate.

### Ti(III) Reformatsky (Cossy)

Replace Zn with Cp₂TiCl₂ / Mn or Zn → enables
cyclopropanation + intramolecular aldol-type chemistry.

### Asymmetric Reformatsky

Cossy + Trost developed chiral catalyst variants — Zn /
chiral diamines, or Zn / BINOL catalysts → > 90 % ee for
β-hydroxy esters.

## Other related zinc-mediated reactions

### Simmons-Smith cyclopropanation

```
RCH=CHR' + CH₂I₂ + Zn(Cu) → cyclopropane
```

Zinc-carbenoid (ZnCH₂I, "carbenoid" because not free
carbene) does syn cyclopropanation. Tolerates many FGs.

Asymmetric: **Furukawa modification** (Et₂Zn + CH₂I₂) +
chiral catalysis.

### Negishi coupling

```
Ar-X + R-ZnX + Pd cat. → Ar-R
```

Uses an organozinc partner. Tolerates many functional
groups (esters, ketones, nitriles, even Bpin).

### Frontier (Ni-catalysed Negishi-like)

Modern: catalyst can be Ni instead of Pd; cheaper. Used
in pharma SAR for sp³-sp³ couplings.

## Failures + alternatives

The Reformatsky has limitations:

- **Hindered ketones** — slow; consider Mukaiyama
  (silyl ketene acetal + Lewis acid).
- **Acid-sensitive substrates** — sometimes incompatible
  with Zn salt by-products.
- **Stereoselectivity** — moderate without chiral catalyst;
  use Evans aldol instead for better de.
