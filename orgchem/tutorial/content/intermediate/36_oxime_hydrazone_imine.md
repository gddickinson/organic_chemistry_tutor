# Oxime, hydrazone, imine chemistry

Carbonyl + amine variants give condensation products that
are workhorse intermediates in organic synthesis.

## Imine (Schiff base)

```
R₂C=O + R'NH₂  ⇌  R₂C=NR' + H₂O
```

- **Acid-catalysed**; equilibrium driven by H₂O removal
  (molecular sieves, Dean-Stark trap).
- Imine pKaH ~ 6-7 (less basic than parent amine).
- Hydrolyses readily back to ketone + amine in aqueous
  acid.

### Reductive amination

```
R₂C=O + R'NH₂ → R₂C=NR' (imine, in situ) + NaBH(OAc)₃ → R₂CH-NHR'
```

The most useful amine synthesis. NaBH(OAc)₃ (STAB) reduces
iminium / iminium hydrate selectively over ketone (because
iminium is more electrophilic). Workhorse for SAR
chemistry — install diverse amines in one step.

### Vinyl imine = enamine

```
R₂CH-CO-R' + R''₂NH ⇌ R₂C=CR'-NR''₂ + H₂O
```

Stork enamine alkylation: enamine acts as soft nucleophile
at α-C (umpolung-like). After alkylation, hydrolyse back
to α-substituted ketone. Avoids over-alkylation that
plagues direct enolate alkylation.

## Hydrazone

```
R₂C=O + H₂N-NHR' → R₂C=N-NHR' + H₂O
```

Hydrazines (H₂N-NH-R) condense like amines but the
resulting hydrazone has unique reactivity:

- **Wolff-Kishner reduction** — heated hydrazone + KOH →
  diazo intermediate → loses N₂ + gains H₂ → C=O reduced
  to CH₂.
- **Bamford-Stevens / Shapiro** — hydrazone + n-BuLi →
  in-situ diazo → carbene → alkene.

## Tosylhydrazone (Bamford-Stevens / Shapiro)

```
R₂C=O + TsNHNH₂ → R₂C=N-NHTs (tosylhydrazone)
                  + 2 eq n-BuLi → vinyllithium → quench
```

Shapiro: vinyllithium can be trapped with electrophiles
(D₂O for vinyl-D, RCHO for allylic alcohol, etc.).

## Oxime

```
R₂C=O + H₂N-OH → R₂C=N-OH + H₂O
```

- **Oximes** are characteristic crystalline derivatives —
  classical structure-confirmation tool (each ketone has
  a unique mp).
- E + Z isomers possible; often interconvert thermally.

### Beckmann rearrangement

```
R₂C=N-OH + acid (or PCl₅, SOCl₂, conc. H₂SO₄) →
R-CO-NHR'        (where R' was anti to OH in the oxime)
```

The group **anti to OH** migrates to nitrogen with loss of
water → amide. Mechanism: O leaves as a leaving group →
trans-migration of the anti-substituent to make a nitrilium
ion → water attacks → amide.

Industrial example: cyclohexanone oxime → caprolactam (by
Beckmann + cyclisation) → polymer Nylon-6.

## Semicarbazone

```
R₂C=O + H₂N-NH-CO-NH₂ → R₂C=N-NH-CO-NH₂ + H₂O
```

Like hydrazone but with a urea tail. Used historically as
a crystalline derivative for ketone identification + as
a protecting group for ketones.

## DNPH (2,4-dinitrophenylhydrazine)

```
R₂C=O + 2,4-(NO₂)₂C₆H₃NH-NH₂ → 2,4-(NO₂)₂C₆H₃-NH-N=CR₂ (yellow-orange)
```

Classical qualitative test for ketones / aldehydes; gives
brightly coloured precipitate; mp distinguishes between
ketones.

## Aza-Wittig

```
R-N=PPh₃ + R'₂C=O → R-N=CR'₂ + Ph₃P=O
```

The "aza" version of Wittig — installs C=N instead of
C=C. Useful when the imine isn't easy to form by direct
condensation.

## Modern asymmetric chemistry

- **Ellman's chiral N-tert-butanesulfinamide** (1997)
  → chiral imines → diastereoselective addition of
  Grignards / hydrides → enantioenriched amines after
  acid hydrolysis.
- **Jacobsen / Doyle thiourea + chiral imine** → asymmetric
  Strecker, Mannich, hydrocyanation.
