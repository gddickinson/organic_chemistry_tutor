# Membrane trafficking + secretion

Eukaryotic cells move proteins, lipids, + cargo between
membrane compartments via vesicles. The trafficking system
is one of the most molecularly-detailed bodies of cell
biology + the substrate for endocrine secretion, neuro-
transmission, immune-cell function, and viral entry.

## The endomembrane system

Compartments connected by vesicular flow:

- **Endoplasmic reticulum (ER)** — protein synthesis +
  folding + early glycosylation (rough ER); lipid
  synthesis + Ca²⁺ storage + detox (smooth ER).
- **Golgi apparatus** — cis-, medial-, trans-cisternae +
  trans-Golgi network (TGN). Protein modification
  (terminal glycosylation, sulfation, O-glycosylation,
  proteolytic processing of pro-hormones), sorting.
- **Endosomes** — early (sorting hub) → late → lysosome.
- **Lysosomes** — acidic (pH 4.5-5) degradation.
- **Plasma membrane** — exit + entry.
- **Secretory vesicles** — regulated secretion (insulin
  granules, synaptic vesicles, mast-cell histamine).

Cargo flows generally **anterograde** (ER → Golgi → PM)
but with extensive **retrograde** (PM → endosome → Golgi
→ ER) recycling.

## Vesicle budding + coat proteins

Each vesicle type uses a specific coat protein:

- **COPII** — buds vesicles from ER; carries cargo to
  Golgi. Sec23/24 + Sec13/31 + Sar1 GTPase.
- **COPI** — buds from Golgi; carries cargo retrograde
  Golgi → ER + within Golgi. ARF1 GTPase + 7-subunit
  coatomer.
- **Clathrin** — buds from PM (endocytosis) + TGN
  (sorting to endosomes). Clathrin triskelion + AP1-5
  adaptor complexes + dynamin GTPase for scission.

The coat does three things:
1. Curves the membrane.
2. Selects cargo (via adaptor-cargo interactions).
3. Recruits scission machinery to release the vesicle.

After scission the coat disassembles + the naked vesicle
travels to its target.

## Vesicle targeting + fusion — the SNARE machinery

How does a vesicle find the right destination?

- **Rab GTPases** — > 60 in humans; each compartment has
  its characteristic Rab(s). Rabs recruit tethering
  factors that grab incoming vesicles.
- **Tethering complexes** — long coiled-coil proteins
  (golgins) or multi-subunit complexes (HOPS, CORVET,
  exocyst) that bridge vesicle to target membrane over
  ~ 100 nm distances.
- **SNAREs** (soluble NSF attachment protein receptors) —
  membrane-anchored proteins on both vesicle (v-SNARE,
  e.g. synaptobrevin/VAMP) and target (t-SNAREs, e.g.
  syntaxin + SNAP-25). Form a four-helix coiled coil
  that pulls the two membranes together → fusion.
- **NSF + α-SNAP** — recycle SNAREs after fusion (ATP-
  dependent disassembly of the four-helix bundle).

The SNARE hypothesis (Rothman + Sudhof + Schekman; joint
2013 Nobel) is one of cell biology's most elegant
mechanisms.

## Synaptic-vesicle exocytosis

The ultimate fast-fusion machine:

1. Synaptic vesicles dock at the active zone via
   v-SNARE (VAMP/synaptobrevin) + t-SNAREs (syntaxin,
   SNAP-25), partially zippered.
2. Action potential → Ca²⁺ influx through Cav2.1/2.2.
3. **Synaptotagmin-1** (vesicle-associated Ca²⁺ sensor)
   binds Ca²⁺ → completes SNARE zippering + drives
   fusion in < 1 ms.
4. Neurotransmitter released into synaptic cleft.
5. Endocytosis recovers the vesicle membrane (kiss-and-
   run, or full collapse + clathrin-coated re-uptake).

**Botulinum + tetanus toxin light chains** are zinc
metalloproteases that cleave SNAREs (BoNT-A → SNAP-25,
BoNT-B → VAMP, BoNT-C → syntaxin) → block neuro-
transmitter release. Botulism is flaccid paralysis;
tetanus blocks inhibitory glycine + GABA release in
spinal cord (light chain reaches via retrograde axonal
transport) → unopposed motor firing → spastic paralysis.

## Endocytosis pathways

Multiple uptake routes:

- **Clathrin-mediated endocytosis (CME)** — the
  workhorse. Specific cargo via adaptors (transferrin
  receptor → AP2; LDL receptor → ARH/Dab2). Vesicle
  size ~ 100 nm.
- **Caveolae** — flask-shaped invaginations rich in
  caveolin + cholesterol; specific signal-platform
  function.
- **Macropinocytosis** — large bulk uptake of fluid +
  membrane; growth-factor + Ras-GTPase-driven; major
  route for some pathogens (Salmonella) + dendritic-
  cell antigen sampling.
- **Phagocytosis** — particle-specific (> 0.5 µm); the
  main microbe-clearance mechanism in macrophages +
  neutrophils.

## Secretory pathway specialisations

Different cell types deploy specialised secretory
machinery:

- **Constitutive secretion** — most cells; vesicles
  bud from TGN + immediately fuse with PM. Continuous
  collagen secretion in fibroblasts.
- **Regulated secretion** — pre-formed dense-core
  granules wait for a signal. Insulin granules in β-
  cells (glucose → Ca²⁺ → fusion); chromaffin granules
  in adrenal medulla; mast-cell histamine.
- **Apocrine + holocrine** — non-vesicular release
  modes.

## Pathogen exploitation

Many pathogens hijack vesicular trafficking:

- **Influenza, HIV** enter via clathrin-mediated
  endocytosis or macropinocytosis.
- **Salmonella** induces macropinocytic ruffles via T3SS
  effectors (SopE, SopB) → engulfment.
- **Listeria** escapes phagosome via listeriolysin O →
  intracellular replication.
- **Legionella** establishes a non-fusogenic Legionella-
  containing vacuole via T4SS effectors (~ 300 of them,
  the largest known effector arsenal).
- **Mycobacterium tuberculosis** prevents phagosome-
  lysosome fusion via PtpA + Eis + PknG.

## Diseases of trafficking

- **Lysosomal storage diseases** — single-enzyme
  deficiencies in lysosomal degradation: Tay-Sachs
  (hexosaminidase A), Gaucher (glucocerebrosidase),
  Pompe (acid α-glucosidase), Hunter syndrome (iduronate
  sulfatase). Enzyme replacement therapy for several;
  gene therapy + pharmacological chaperones in
  development.
- **Charcot-Marie-Tooth disease** — peripheral
  neuropathy; some forms (CMT2B, type-7) involve Rab7
  or dynamin-2 mutations affecting axonal trafficking.
- **Niemann-Pick C** — defective intracellular
  cholesterol trafficking.
- **Cystic fibrosis** — ΔF508 CFTR is misfolded + ER-
  retained; modulators (lumacaftor + ivacaftor +
  elexacaftor) rescue trafficking + activity.
