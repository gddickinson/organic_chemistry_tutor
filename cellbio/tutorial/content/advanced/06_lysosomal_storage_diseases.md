# Lysosomal degradation + lysosomal storage diseases

The lysosome digests every category of cellular
macromolecule via ~ 60 acid hydrolases working
together at low pH.  When any one of those
hydrolases is deficient, its substrate accumulates
+ disrupts cell + tissue function — the
lysosomal storage diseases (LSDs), one of the
oldest + best-studied groups of inherited
metabolic diseases.

## Lysosomal architecture

- ~ 0.1-1 µm diameter; pH 4.5-5.0; ~ 60 acid
  hydrolases.
- **V-ATPase** pumps H⁺ INTO lumen; cycle of
  ATP → 1 H⁺ × ~ 8-9 per turnover.
- **LAMP1 + LAMP2 + CD63** glycoproteins protect
  the limiting membrane from luminal hydrolases.
- **Cathepsins** (cysteine + serine + aspartate
  proteases) — D, B, L, S, C, K, F, H, etc.
- **Glycosidases + sulfatases + phosphatases +
  lipases + nucleases** — substrate-specific.

## Hydrolase delivery

Most lysosomal hydrolases reach the lysosome via
the **mannose-6-phosphate (M6P) pathway**:

1. **In ER** — N-linked glycan added during
   translation.
2. **In cis-Golgi** — GlcNAc-1-phosphotransferase
   (encoded by *GNPTAB / GNPTG*) transfers
   GlcNAc-1-P onto specific mannose residues.
3. **In trans-Golgi** — uncovering enzyme exposes
   M6P.
4. **Trans-Golgi-network** — M6P-receptor (CI-MPR
   or CD-MPR) binds M6P-tagged hydrolase.
5. **Clathrin-coated vesicle** delivers to late
   endosome.
6. **Acidic pH releases** hydrolase from receptor;
   receptor recycles back to TGN.

Defects in M6P tagging → I-cell disease + pseudo-
Hurler polydystrophy (mucolipidoses II + III) —
multiple lysosomal-enzyme deficiencies because
none can be tagged + delivered.

## Cargo arrival at the lysosome

Three input streams:

### Endocytosis

- **Clathrin-mediated** — receptor-mediated
  uptake (LDL, transferrin) → early endosome →
  late endosome → MVB → lysosome.
- **Caveolar / clathrin-independent** —
  alternative routes; pathogen + GPI-protein
  trafficking.
- **Macropinocytosis** — bulk fluid uptake;
  growth-factor-driven; antigen sampling by
  DCs.
- **Phagocytosis** — > 0.5 µm particles;
  Fcγ + CR3 + scavenger receptors.

### Autophagy

- Autophagosome fuses with lysosome (STX17 +
  SNAP-29 + VAMP8).
- Inner autophagosome membrane + cargo degraded.
- (Covered in **CB-4.0 intermediate "Autophagy
  + UPS" lesson**.)

### Direct trafficking

- TGN-derived vesicles deliver new lysosomal
  hydrolases (M6P pathway above) + membrane
  proteins (LAMPs).

## Lysosomal storage diseases (LSDs)

> 70 distinct LSDs catalogued.  Combined incidence
~ 1 / 5 000 births.  Each has a defined enzyme
deficiency + accumulating substrate + tissue
distribution + clinical phenotype.

Categorised by substrate class:

### Sphingolipidoses

| Disease | Enzyme deficient | Substrate | Therapy |
|---------|-------------------|-----------|---------|
| Tay-Sachs | β-Hexosaminidase A | GM2 ganglioside | Supportive |
| Sandhoff | β-Hexosaminidase A + B | GM2 | Supportive |
| Gaucher | Glucocerebrosidase (GBA) | Glucocerebroside | ERT (imiglucerase / velaglucerase / taliglucerase); SRT (miglustat / eliglustat) |
| Niemann-Pick A + B | Acid sphingomyelinase | Sphingomyelin | ERT (olipudase alfa, 2022) |
| Niemann-Pick C | NPC1 / NPC2 (cholesterol transport) | Cholesterol + sphingolipids | Miglustat |
| Fabry | α-Galactosidase A | Globotriaosylceramide (Gb3) | ERT (agalsidase α / β); migalastat (chaperone); pegunigalsidase alfa; ERT-replacement gene therapy |
| Krabbe | Galactocerebrosidase | Galactocerebroside | HSCT |
| Metachromatic leukodystrophy | Arylsulfatase A | Sulfatides | HSCT; gene therapy (Libmeldy, atidarsagene autotemcel) |

### Mucopolysaccharidoses (MPS)

GAG-degradation defects:

| MPS | Enzyme | Substrate | Therapy |
|-----|--------|-----------|---------|
| MPS I (Hurler / Scheie) | α-L-Iduronidase | DS + HS | ERT (laronidase); HSCT |
| MPS II (Hunter) | Iduronate-2-sulfatase | DS + HS | ERT (idursulfase); HSCT |
| MPS III (Sanfilippo, A-D) | Heparan-N-sulfatase / NAGLU / HGSNAT / GNS | HS | Gene therapy in trials |
| MPS IV (Morquio A + B) | GALNS / GLB1 | KS / KS+CS | ERT (elosulfase α for IVA) |
| MPS VI (Maroteaux-Lamy) | Arylsulfatase B | DS | ERT (galsulfase) |
| MPS VII (Sly) | β-Glucuronidase | DS, HS, CS | ERT (vestronidase α) |
| MPS IX | Hyaluronidase | HA | Rare |

### Glycogen storage (lysosomal)

- **Pompe disease** — acid α-glucosidase (GAA)
  deficiency → glycogen accumulation; cardiac +
  skeletal-muscle myopathy.  ERT (alglucosidase α
  / avalglucosidase α / cipaglucosidase α + miglustat).

### Mucolipidoses

- **ML I (Sialidosis)** — neuraminidase deficiency.
- **ML II (I-cell) + ML III** — GNPTAB / GNPTG
  defects; multiple lysosomal-enzyme mis-routing.
- **ML IV** — MCOLN1 (TRPML1, lysosomal Ca²⁺ +
  TRPML1 channel) deficiency.

### Glycoproteinoses

- **α + β-Mannosidoses** — α-mannosidase / β-
  mannosidase.
- **Fucosidosis** — α-fucosidase.
- **Aspartylglucosaminuria** — AGA.
- **Schindler / Kanzaki** — α-NAGA.

### Neuronal ceroid lipofuscinoses (NCLs)

Heterogeneous group; ~ 14 genes (CLN1 - CLN14).
Childhood-onset blindness + seizures + dementia
+ motor decline.

- **CLN1 (PPT1)** — palmitoyl-protein thioesterase.
- **CLN2 (TPP1)** — tripeptidyl peptidase 1; ERT
  (cerliponase α intracerebroventricular,
  approved 2017).
- **CLN3** — Batten disease; transmembrane lysosomal
  protein.

### Cystinosis

- Cystinosin (CTNS) lysosomal cystine transporter
  deficiency → cystine crystals in lysosomes.
- Cysteamine therapy depletes intra-lysosomal
  cystine.

### Cholesterol-trafficking LSDs

- **Niemann-Pick C (NPC1 / NPC2)** — cholesterol
  + sphingolipid accumulation.  Miglustat slows
  progression; arimoclomol + adrabetadex in
  trials.
- **Wolman / CESD (Cholesterol ester storage
  disease)** — lysosomal acid lipase (LIPA)
  deficiency.  ERT (sebelipase α).

## Lysosomal-enzyme replacement therapy (ERT)

Pioneered by Roscoe Brady's Gaucher work in the
1970s + commercial Cerezyme (1991).

Concept: infuse the missing enzyme intravenously;
M6P-tagged enzyme is taken up by cells via the
CI-MPR receptor + delivered to lysosomes.

Limitations:
- Doesn't cross the blood-brain barrier → no
  CNS efficacy in CNS-affecting LSDs.
- Frequent infusions needed (weekly to bi-weekly).
- Anti-drug antibody development.
- Very expensive (~ $200K-500K / year).

CNS-targeting variants:
- **Intracerebroventricular ERT** (cerliponase α
  for CLN2; reduces neurological decline).
- **Receptor-mediated transcytosis** (Aldevron
  + Sanofi etopepim alfa using transferrin-
  receptor-binding for MPS IIIA — Phase 3).
- **Engineered enzymes with BBB-penetrating tags**
  (e.g. brain-targeted fusion proteins, ANGIOPEP-2
  conjugates).

## Substrate reduction therapy (SRT)

Reduce substrate synthesis instead of restoring
degradation:
- **Miglustat (Zavesca)** — glucosylceramide-
  synthase inhibitor; Gaucher type 1 + Niemann-
  Pick C.
- **Eliglustat (Cerdelga)** — newer + better-
  tolerated; Gaucher type 1.
- **Venglustat** — in trials for Gaucher + GBA-
  associated PD.

## Pharmacological chaperone therapy

Small molecules that bind misfolded enzymes +
stabilise + traffic them to the lysosome:
- **Migalastat (Galafold)** — Fabry disease (for
  GLA mutations responsive to chaperone).
- **Ambroxol** — Gaucher (off-label) + GBA-
  associated PD trials.
- **NCGC00188758** + others in development.

## Gene therapy

Increasingly major for LSDs:

- **Libmeldy / Atidarsagene autotemcel** — ex-vivo
  lentiviral gene therapy for metachromatic
  leukodystrophy (HSC-targeted; ARSA gene); EU
  approved 2020, US 2024.
- **Skysona / Elivaldogene autotemcel** — X-linked
  adrenoleukodystrophy.
- **AAV-based in-vivo gene therapy** — multiple
  LSDs in clinical trials (MPS I, II, III; CLN2;
  Pompe; Fabry; Niemann-Pick A).
- **Roctavian / Hemgenix** — haemophilias (not LSD
  but similar AAV technology).

## Bone-marrow / haematopoietic stem-cell
transplant (HSCT)

- Donor cells produce normal enzyme that can be
  taken up by recipient cells via M6P pathway
  (cross-correction).
- Effective for some LSDs:
  - MPS I (Hurler) — best if early (< 2 years).
  - MPS VI + VII — variable.
  - Krabbe (early in life).
  - Metachromatic leukodystrophy.
- Limited for CNS-only disease (modest benefit).

## Lysosomes as signalling hubs

A 2010s+ paradigm shift: lysosomes signal, not
just degrade.

### mTORC1 activation

- mTORC1 activates on the lysosomal surface in
  response to amino acids (Sabatini lab):
  - **Rag GTPases (RagA/B + RagC/D)** anchored
    to lysosome via Ragulator complex.
  - Amino-acid sufficiency → Rag activation →
    recruits mTORC1 to lysosome → Rheb-GTP
    activates mTORC1 kinase activity.
  - **Sestrin2 + Castor + SAMTOR** — leucine,
    arginine, methionine sensors.
  - **GATOR1 + GATOR2 + KICSTOR** — regulators.
- mTORC1 phosphorylates S6K1 + 4E-BP1 + ULK1
  + TFEB → translation + autophagy + lysosomal
  biogenesis control.

### TFEB

- Master transcription factor for lysosomal +
  autophagy genes (CLEAR network).
- mTORC1 phosphorylates TFEB on lysosome → 14-3-3
  binding → cytosolic retention.
- Starvation / lysosomal stress → mTORC1
  inactivation → TFEB dephosphorylated →
  nuclear translocation → upregulates lysosomal
  + autophagy + lipid-metabolism genes.

### Lysosomal Ca²⁺

- Lysosomal lumen contains Ca²⁺ (~ 500 µM);
  released via TPC1 + TPC2 (NAADP-activated) +
  TRPML1 (PI(3,5)P₂-activated).
- Ca²⁺ release activates calcineurin → TFEB
  dephosphorylation → autophagy upregulation.

### Lysosomal cholesterol sensing

- Lysosome-resident NPC1 + LYCHOS (newly named;
  formerly KIAA1024) sense cholesterol.
- Couples to mTORC1 + SREBP signalling.

## Lysosomal pH-sensing technology

- **LysoTracker, LysoSensor** — pH-sensitive
  dyes for live imaging.
- **pHluorins + fluorescent pH-sensors** —
  genetically encoded.
- **TFEB-GFP nuclear-translocation assays** —
  lysosomal-stress reporters.
- **Cathepsin-cleavable fluorogenic substrates**
  (Magic Red, BMV109) — activity assays.
